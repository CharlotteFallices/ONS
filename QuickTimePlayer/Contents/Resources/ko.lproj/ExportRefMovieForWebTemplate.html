사용자의 웹 사이트에 $(TITLE)을(를) 올리기 위한 지침

QuickTime Pro를 사용하여 $(TITLE)을(를) 보내기 하였습니다. 다음의 정보가 이 동영상을 사용자의 웹 사이트로 올리는 것을 도와줍니다.

이 HTML 지침 파일이 있는 동일한 폴더에, QuickTime Pro에서 생성한 웹 전송에 대해 최적화된 다양한 버전의 사용자 동영상이 있습니다. 또한 해당 폴더에는 사용자의 다양한 장비 및 연결 속도에 대해 최상의 버전으로 선택한 참조 동영상도 들어 있습니다. 이 참조 동영상을 사용자의 HTML 페이지에 포함시켜, 사용자의 사이트에 적절한 비디오 품질을 보장할 수 있습니다.

미리보기입니다(인터넷 연결 필요).

다음 HTML 코드를 사용하여 동영상을 사용자의 웹 페이지로 올리십시오.

중요한 알림:

  • 사용자의 미디어를 인터넷에서 보려면, 미디어 항목(동영상, 포스터 이미지)이 공용 연결 서버에 있어야 합니다.
  • 미디어 항목이 사용자의 웹 페이지와 다른 위치에서 제공되었다면 해당 URL 매개변수를 수정해야 합니다.
  • 동영상 URL 매개변수는 항상 소스 동영상이 아니라 참조 동영상 ($(REF_MOVIE_HTML_ESCAPED))을 가리킵니다.
  • 참조 동영상은 소스 동영상과 동일한 폴더에 있어야 합니다.
  • 소스 동영상 이름은 변경할 수 없지만, 참조 동영상의 이름은 변경할 수 있습니다.(HTML 코드에서도 해당 이름을 변경해야 합니다)
  • XHTML을 적절하게 유효화하려면 <style> 태그 및 첫번째 <script> 태그가 사용자 페이지의 <head> 태그에 있어야 합니다.

사용자 웹 페이지 <head>에 이 코드를 사용하십시오.

사용자 웹 페이지 <body>에 이 코드를 사용하십시오.

보내기한 항목:

  • 참조 동영상:
    • $(REF_MOVIE_HTML_ESCAPED)
  • 포스터 프레임:
    • $(POSTER_IMAGE_HTML_ESCAPED)
  • 버전:
      $(EXPORT_NAMES)
