Instruções para publicação de $(TITLE) no seu website

Exportou $(TITLE) através do QuickTime Pro. A seguinte informação irá ajudá-lo a publicar este filme no seu website.

Na mesma pasta em que se encontra este ficheiro HTML de instruções, o QuickTime Pro colocou várias versões do seu filme optimizado para distribuição na Web. Também nessa pasta existe um filme de referência, que selecciona a melhor versão para os múltiplos dispositivos e velocidades de ligação dos utilizadores. Ao incluir este filme de referência na sua página HTML, pode garantir uma experiência de vídeo de qualidade no seu website.

Notas importantes:

  • Para tornar os seus ficheiros multimédia visíveis na Internet, os recursos multimédia (filmes, imagem de cartaz) têm de ser colocados num servidor acessível ao público.
  • Os parâmetros URL têm de ser modificados, caso os seus recursos multimédia sejam servidos a partir de uma localização diferente da página web.
  • Os parâmetros URL do filme têm de apontar sempre para o filme de referência ($(REF_MOVIE_HTML_ESCAPED)) em vez de para um filme de origem.
  • O filme de referência tem de ser guardado na mesma pasta que os filmes de origem.
  • Os nomes dos filmes de origem não podem ser alterados, mas pode modificar o nome do filme de referência (não se esqueça de alterar também o nome no código HTML).
  • Atente que, para validação correcta como XHTML, a etiqueta <style> e a primeira etiqueta <script> deverão ser colocadas dentro da etiqueta <head> da página.
