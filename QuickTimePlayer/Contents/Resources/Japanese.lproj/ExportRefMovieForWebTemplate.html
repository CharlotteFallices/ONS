$(TITLE) を Web サイトにポストする方法

$(TITLE) は、QuickTime Pro を使って書き出されました。以下は、このムービーをご自分の Web サイトにポストする際に役に立つ情報です。

この HTML ファイルと同じフォルダに、Web 経由の配信に最適化されたさまざまなバージョンのムービーが置かれています。また、同じフォルダに、ユーザの方が使用しているデバイスや接続速度に最適なバージョンを選択する参照ムービーも含まれています。ご自分の HTML ページにこの参照ムービーを含めることによって、サイトで質の高いビデオ機能を提供できます。

ムービーをご自分の Web ページに埋め込むときは、以下の HTML コードをお使いください。

重要なご注意：

  • メディアをインターネット上で視聴できるようにするには、メディア・アセット（ムービーやポスターイメージ）をだれでもアクセスできるサーバ上に置く必要があります。
  • お使いのメディア・アセットがご自分の Web ページとは異なる場所にある場合、URL のパラメータを変更する必要があります。
  • ムービーの URL のパラメータは、ソースムービーではなく、必ず、参照ムービー（$(REF_MOVIE_HTML_ESCAPED)）をポイントする必要があります。
  • 参照ムービーは、ソースムービーと同じフォルダ内に保存する必要があります。
  • ソースムービーの名前は変更できませんが、参照ムービーの名前は変更できます（HTML コードでも名前を変更するのを忘れないでください）。
  • XHTML として有効にするには、<style> タグと最初の <script> タグをページの <head> タグ内に配置する必要があることに注意してください。
