QuickTime Player Help

Entering Your QuickTime 7 Pro Registration Code

If you have a valid QuickTime 7 Pro registration code already on your computer, QuickTime Player 7 recognizes the registration code and enables QuickTime Pro features.

Otherwise, to enter a QuickTime 7 Pro registration code:

  1. Choose QuickTime Player 7 > Registration.
  2. Enter your QuickTime 7 Pro registration code.
