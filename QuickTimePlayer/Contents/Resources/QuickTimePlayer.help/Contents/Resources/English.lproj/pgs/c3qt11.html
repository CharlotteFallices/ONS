Adding a Text Track to a QuickTime Movie

You can import a text file into a QuickTime movie.

To add a text track:

  1. Create a text file containing the text you want in the movie, and save the file as text only (.txt).

    Separate each paragraph with a Carriage Return character. Each paragraph appears in a separate frame of the movie. By default, each text frame is displayed for 2 seconds.

  2. In QuickTime Player 7, choose File > Open File and select the text file.
  3. Click Open.
