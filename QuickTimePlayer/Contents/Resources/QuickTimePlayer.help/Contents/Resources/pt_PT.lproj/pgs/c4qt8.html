Como preparar um filme para o início rápido

É possível configurar um filme para começar a ser reproduzido a partir de um servidor web, antes de ser completamente descarregado para o disco rígido do utilizador. Isto chama-se um filme de "Início rápido".

Defina a opção "Início rápido" mesmo antes de distribuir o filme; ao realizar outras alterações e guardar novamente o filme poderá anular esta definição.

Se o filme estiver no formato que pretende, pode defini-lo para um "Início rápido", seleccionando Ficheiro > "Guardar como" e depois "Guardar como filme autónomo". Se o filme não tiver o formato pretendido, será necessário codificá-lo executando os passos seguintes.

Para converter um filme e configurá-lo para "Início rápido":

  1. No QuickTime Player, seleccione Ficheiro > Exportar.
  2. Seleccione "Filme para filme QuickTime" a partir do menu instantâneo Exportar.
  3. Clique em Opções e seleccione as opções de compressão de vídeo e som apropriadas para transmissão pela web.

    Para obter mais informações, consulte Como personalizar as definições de exportação de vídeo, Como personalizar as definições de exportação de áudio.

  4. Certifique-se de que a opção "Preparar para transmissão na Internet" está seleccionada e de que "Início rápido" aparece no menu instantâneo.
