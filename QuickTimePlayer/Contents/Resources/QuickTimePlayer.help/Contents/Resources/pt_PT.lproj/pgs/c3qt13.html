Como especificar estilos de tipos de letra para faixas de texto

É possível alterar os atributos do tipo de letra de uma faixa de texto que já está num filme. Para tal, exporte a faixa, efectue as alterações e volte a reimportar a faixa.

Para alterar os estilos do tipo de letra da faixa de texto:

  1. No QuickTime Player, seleccione Ficheiro > Exportar.
  2. Seleccione "Texto para Texto" a partir do menu instantâneo Exportar.
  3. Clique em Opções e seleccione "Mostrar texto, descritores e hora".
  4. Utilize um editor de texto para alterar os valores após o identificador {QTtext}.
  5. Volte a importar a faixa de texto para o seu filme.

Também pode inserir identificadores ao longo do texto para alterar os valores durante a reprodução do filme.
