Ajuda do QuickTime Player

Adicionar uma faixa de texto a um filme QuickTime

É possível importar um ficheiro de texto para um filme QuickTime.

Para adicionar uma faixa de texto:

  1. Crie um ficheiro de texto com o texto que pretende que apareça no filme e guarde-o como só texto (.txt).

    Separe cada parágrafo por um carácter de marca de retorno. Cada parágrafo aparece num fotograma separado do filme. Por predefinição, cada fotograma de texto é exibido durante 2 segundos.

  2. No QuickTime Player, seleccione Ficheiro > Abrir ficheiro e seleccione o ficheiro de texto.
  3. Clique em Abrir.
