Como definir o fotograma de cartaz de um filme

Um "fotograma de cartaz" é uma imagem estática de um filme que representa o filme em locais como a janela de Favoritos do QuickTime Player e o Finder. O fotograma de cartaz predefinido é o primeiro fotograma do filme. É possível alterar o fotograma de cartaz.

Para alterar o fotograma de cartaz de um filme:

  1. No QuickTime Player, arraste o cursor de reprodução (ou utilize as teclas das setas para mover o cursor de reprodução) para o fotograma pretendido.
  2. Seleccione Visualização > Definir fotograma de cartaz.

Para ver o fotograma de cartaz de um filme, seleccione Visualização > "Ir para o fotograma de cartaz".
