Como especificar idiomas para faixas individuais

É possível criar um filme com suporte para vários idiomas, especificando um idioma para uma faixa individual. Por exemplo, pode criar um filme que contenha uma faixa de áudio separada para cada idioma. O QuickTime reproduz automaticamente a faixa correcta, com base na definição de idioma do computador do utilizador.

Para especificar idiomas:

  1. Crie ou compile as faixas alternativas.

    Grave a voz-off em cada um dos idiomas pretendidos ou crie faixas de texto para utilizar como legendas. Guarde cada faixa alternativa como um filme autónomo e coloque todos os idiomas juntos, numa única pasta.

  2. Coloque uma cópia do seu filme original, autónomo e excluindo quaisquer faixas que agora estão guardadas como alternativas, na pasta que as contém.
  3. Abra o filme e seleccione Janela > Mostrar propriedades do filme.
  4. Na janela de propriedades, seleccione uma faixa de som ou de texto e clique em "Outras definições".
  5. Seleccione o idioma para esta faixa a partir do menu instantâneo Idioma.
  6. Seleccione uma faixa para utilizar quando for seleccionado um idioma diferente a partir do menu instantâneo Alternativa.
  7. Guarde como um filme autónomo (seleccione Ficheiro > "Guardar como" e certifique-se de que a opção "Tornar filme autónomo" está seleccionada).
  8. Repita estes passos para as faixas de idiomas adicionais; de cada vez que o fizer, seleccione o idioma anteriormente seleccionado a partir do menu instantâneo Alternativa.
