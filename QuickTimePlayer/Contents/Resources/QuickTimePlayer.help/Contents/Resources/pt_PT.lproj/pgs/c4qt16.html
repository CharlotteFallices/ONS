Opções de exportação para o formato de ficheiro 3G

Depois de seleccionar Ficheiro > Exportar e clicar em Opções, os dois primeiros itens do menu instantâneo "Formato do ficheiro" são os formatos standard.

  • 3GPP é para redes GSM. Com este formato, pode utilizar:
    • MPEG-4, H.263 ou vídeo H.264
    • AAC ou áudio AMR
    • Texto temporizado 3G
  • 3GPP2 é para redes CDMA 2000. Com este formato, pode utilizar:
    • MPEG-4, H.263 ou vídeo H.264
    • AAC, AMR ou áudio QCELP
    • Texto temporizado 3G
    • Fragmentos de filmes (permite que a reprodução de filmes mais longos comece mais cedo, uma vez que apenas o fragmento, não o filme inteiro, tem de caber no telemóvel)

Os restantes formatos são fornecidos para redes específicas. Com estes formatos, pode restringir a distribuição de modo a que um ficheiro não possa ser partilhado a partir do telemóvel do destinatário (defina esta opção no painel Avançadas). Estes formatos podem limitar o tamanho de ficheiro ou a velocidade de dados aceitável; contacte o fornecedor de serviços para mais informações.

  • 3GPP (Mobile MP4) é para o serviço "i-motion" 3G da NTT DoCoMo. Com este formato, é possível utilizar as opções 3GPP descritas acima.
  • 3GPP2 (EZmovie) é para o serviço de rede 3G da KDDI. Este formato disponibiliza as mesmas opções do que o 3GPP2 (descritas acima), excepto relativamente ao suporte de áudio AMR.
  • AMC (EZmovie) é para subscritores da KDDI com telemóveis compatíveis com AMC. Com este formato, pode utilizar:
    • Vídeo MPEG-4
    • Áudio QCELP
    • Formato de texto KDDI
