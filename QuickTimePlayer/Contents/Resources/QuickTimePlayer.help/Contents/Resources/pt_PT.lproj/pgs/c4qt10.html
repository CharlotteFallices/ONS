Como criar filmes de referência para optimizar a reprodução na web

Pode utilizar filmes de referência para fornecer o filme adequado à velocidade de ligação de diferentes utilizadores. Por exemplo, pode fazer o streaming de um filme mais pequeno para utilizadores ligados através de um modem standard e de uma versão maior do filme para utilizadores com ligação de banda larga.

A Apple fornece uma ferramenta gratuita, o MakeRefMovie X, que simplifica bastante o processo de criação de filmes de referência. Pode descarregar este software em Apple Developer Connection.
