Como preparar filmes para transmissão pela Internet

É possível criar filmes para distribuição pela Internet. Pode transmitir um filme pela Internet de duas formas:

  • Através de descarregamento HTTP, o filme é descarregado para o disco rígido do cliente. "Início rápido" é uma funcionalidade do QuickTime que permite aos utilizadores ver ou ouvir conteúdos multimédia à medida que estes estão a ser descarregados (muito tempo antes de todo o filme ter sido descarregado) de um servidor web standard para os respectivos discos rígidos. O "Início rápido" funciona bem para filmes pequenos, cujo tamanho de ficheiro é limitado. Assegura uma reprodução de alta qualidade, independentemente das velocidades de ligação dos utilizadores, embora aqueles que possuam ligações mais lentas tenham de aguardar mais tempo até aos conteúdos multimédia começarem a ser reproduzidos.
  • O streaming em tempo real fornecido pelo QuickTime Streaming Server (QTSS) proporciona conteúdos multimédia em tempo real pela Internet, a velocidades desde as de um modem standard até à banda larga. Não chega a haver descarregamento de ficheiros para o disco rígido do utilizador. Os conteúdos multimédia são reproduzidos, mas não guardados, pelo software cliente à medida que são transmitidos. O streaming em tempo real é preferível ao "Início rápido" no caso de webcasts de eventos em directo, envio de vídeos longos, canais de rádio e TV de transmissão contínua na Internet, bem como noutros casos em que não pretenda que os ficheiros sejam guardados no disco rígido do utilizador. O QTSS utiliza o protocolo RTSP.
