Como preparar um filme para streaming em tempo real

É possível configurar um filme para ser transmitido como stream pela Internet. Para tal, deverá comprimir o filme de modo a que a taxa de dados seja apropriada à largura de banda da ligação dos utilizadores.

O formato "Transmissão em sequência sugerida" deve ser utilizado com o QuickTime Streaming Server ou o Darwin Streaming Server. Quando opta pela transmissão em sequência sugerida, são adicionadas ao filme "faixas de sugestões" (informações necessárias para fazer streaming do filme). Se o filme já estiver no formato pretendido, pode prepará-lo para ser transmitido como stream seleccionando Ficheiro > Exportar, e depois "Filme para filme sugerido". Se pretender alterar o formato do filme, efectue os seguintes procedimentos.

Para configurar um filme para streaming pela Internet:

  1. No QuickTime Player, seleccione Ficheiro > Exportar.
  2. Seleccione "Filme para filme QuickTime" a partir do menu instantâneo Exportar.
  3. Seleccione uma opção de streaming a partir do menu instantâneo Usar.
  4. Clique em Opções e seleccione a opção "Preparar para transmissão na Internet".
  5. Seleccione "Transmissão em sequência sugerida" a partir do menu instantâneo.

Se pretender transmitir em streaming um filme QuickTime utilizando o QuickTime Streaming Server, deverá utilizar um compressor de vídeo ou de áudio optimizado para a web para comprimir o ficheiro de filme. Todos os compressores QuickTime são compatíveis com o QuickTime Streaming Server, mas os compressores seguidamente listados estão optimizados para proporcionar os melhores resultados em termos de transmissão pela Internet. Para obter uma lista completa dos compressores suportados pelo QuickTime, consulte a área de Produtos do website do QuickTime (www.apple.com/pt/quicktime/).

  • Compressores de vídeo: H.264, MPEG-4, Sorenson Video (qualquer versão), H.263, H.261
  • Compressores de áudio: AAC, QUALCOMM PureVoice, AMR, QDesign
