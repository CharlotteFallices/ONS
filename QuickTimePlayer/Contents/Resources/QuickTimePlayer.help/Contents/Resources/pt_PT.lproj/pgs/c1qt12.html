Ajuda do QuickTime Player

Como alterar as preferências de reprodução

É possível alterar as preferências de reprodução, escolhendo QuickTime Player > Preferências. As opções disponíveis incluem:

  • Abrir filmes em novos leitores: seleccione para abrir os filmes numa nova janela do leitor; anule a selecção para substituir o filme actualmente na janela do leitor pelo filme novo.
  • Reproduzir automaticamente os filmes ao abrir: seleccione para fazer com que todos os filmes comecem a ser reproduzidos quando são abertos; anule a selecção para que o filme comece apenas quando clicar no botão de reprodução.
  • Utilizar as definições de vídeo de alta qualidade, quando disponíveis: se esta opção for seleccionada, o vídeo fica muito mais nítido e regular, mas é utilizada mais capacidade do processador.
  • Só reproduzir o som do leitor cuja janela está activa: seleccione para reproduzir o áudio de apenas uma janela do QuickTime Player (a janela activa); anule a selecção para reproduzir simultaneamente o som de todos os filmes em reprodução.
  • Reproduzir som quando a aplicação está em segundo plano: seleccione para reproduzir áudio, mesmo que o QuickTime não seja a aplicação activa; anule a selecção para silenciar o áudio quando o QuickTime não é a aplicação activa.
  • Mostrar equalizador: seleccione ou anule a selecção para mostrar ou ocultar o equalizador (que mostra a presença de áudio) na janela do QuickTime Player.
  • Mostrar automaticamente o guia de conteúdo: seleccione para ver a mais recente listagem de conteúdo quando o QuickTime Player abre pela primeira vez (a não ser que abra um ficheiro de filme específico).
  • Fazer pausa nos filmes antes de mudar de utilizador: seleccione para efectuar automaticamente uma pausa num filme quando utiliza a mudança rápida de utilizador, a fim de deixar outra pessoa utilizar o seu computador.
  • Ocultar os indicadores de selecção quando a selecção está vazia: seleccione para remover os indicadores de selecção da linha de tempo, quando não estão seleccionados fotogramas.
  • Mostrar legendagem fechada, quando disponível: seleccione para ver as informações de legendas fechadas, se tiverem sido fornecidas legendas pelos criadores dos filmes.
