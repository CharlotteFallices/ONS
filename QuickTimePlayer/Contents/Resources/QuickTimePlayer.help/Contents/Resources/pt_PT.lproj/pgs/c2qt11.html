Como criar uma imagem estática a partir de um filme

Com o QuickTime Player, é possível exportar um único fotograma de um filme como um ficheiro de imagem.

Para guardar uma imagem estática a partir de um filme:

  1. Desloque o cursor de reprodução para o fotograma que pretende exportar.
  2. Seleccione Ficheiro > Exportar.
  3. Seleccione "Filme para imagem" a partir do menu instantâneo Exportar.
  4. Clique em Opções e seleccione um formato a partir do menu instantâneo "Tipo de compressão".

Para exportar todos os fotogramas do filme como uma série de imagens, seleccione "Filme para sequência de imagens". A exportação de todos os fotogramas pode gerar um número muito grande de ficheiros; deverá optar por exportar para uma pasta e não para a secretária.
