Ajuda do QuickTime Player

Opções de streaming 3G

Quando exporta um filme para o formato 3G (seleccionando Ficheiro > Exportar, e depois "Filme para 3G"), acede às seguintes opções, clicando em Opções e seleccionando Streaming.

Para criar um ficheiro para streaming RTSP para o QuickTime Player, seleccione "Activar streaming". Esta opção cria uma faixa de sugestões (instruções necessárias para fazer streaming de um ficheiro).

"Optimizar para servidor" ajuda o servidor a processar o ficheiro mais rapidamente, mas aumenta o tamanho do ficheiro.
