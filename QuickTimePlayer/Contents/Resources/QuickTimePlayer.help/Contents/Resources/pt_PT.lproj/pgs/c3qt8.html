Como alterar os níveis de volume da faixa de som e outras definições de áudio

É possível alterar o volume e o balanço das faixas de áudio e música. Por exemplo, se um filme tiver mais do que uma faixa de áudio, é possível ajustar o volume das faixas umas relativamente às outras. Também é possível alterar os níveis de graves e agudos.

Para alterar as definições de uma faixa de áudio:

  1. No QuickTime Player, seleccione Janela > Mostrar propriedades do filme.
  2. Na janela de Propriedades, seleccione a faixa de áudio e clique em "Definições de áudio".
  3. Arraste os níveis para ajustar as definições.
  4. Utilize o menu instantâneo à direita de cada canal de áudio para especificar para onde deve ser dirigido o som do canal.

    Para dirigir o som de um canal para o subwoofer, seleccione "Ecrã LFE". Para especificar uma saída em particular para o dispositivo de áudio, seleccione umas das opções Discreto. Para silenciar um canal, seleccione "Não utilizado".

  5. Para desactivar o áudio (sem ter de eliminar a faixa), seleccione Silencioso. Para reproduzir apenas a faixa de áudio seleccionada, seleccione Solo.
  6. Para guardar as definições de balanço e canal com o seu filme, seleccione Ficheiro > Guardar ou "Guardar como".
