Ajuda do QuickTime Player

Como utilizar os números de fotogramas e códigos horários

Quando está a ver um filme no QuickTime Player, pode exibir o tempo do filme actual, os números de fotogramas ou o código horário SMPTE, se o filme tiver uma faixa de código horário (algumas aplicações, como o Final Cut Pro, adicionam automaticamente um código horário durante a captura).

Como apresentar os números de fotogramas, o tempo do filme ou o código horário

A área junto à linha de tempo no QuickTime Player apresenta as informações de tempo do filme actual.

  1. Clique na área de apresentação do tempo.
  2. Seleccione uma opção do menu instantâneo que aparece:

    Standard: Apresenta o tempo do filme no formato HH:MM:SS.

    Código horário: Apresenta o código horário do filme no formato HH:MM:SS:Fotograma. Será exibido o código horário ou código horário "drop-frame", dependendo dos fotogramas por segundo do filme. Dois pontos (:) no segmento dos fotogramas da notação horária indicam que está ser apresentado um código horário "non-drop". Ponto e vírgula (;) indica um código horário "drop-frame". Se o filme tiver uma faixa de código horário, esta será utilizada. Caso contrário, o código horário será calculado a partir de 00:00:00:00 no início do filme.

    Número de fotograma: Apresenta o número de fotograma, incrementando a partir de 0 no início do filme. Um "f" na área de apresentação do tempo indica que a contagem de fotogramas está a ser exibida.
