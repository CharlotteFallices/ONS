Ajuda do QuickTime Player

Como exportar ficheiros utilizando predefinições

O QuickTime Player oferece diversas definições e opções para comprimir vídeo e áudio durante a exportação. Para ajudar a simplificar o processo de comprimir e exportar, o QuickTime Player também oferece predefinições de exportação. Estão disponíveis diferentes predefinições para cada formato de ficheiro.

Para exportar um filme utilizando predefinições:

  1. No QuickTime Player, abra o filme que pretende exportar.
  2. Seleccione Ficheiro > Exportar.
  3. Seleccione um formato de ficheiro a partir do menu instantâneo Exportar.
  4. Seleccione a predefinição que melhor se adequa às suas necessidade a partir do menu instantâneo Usar.

    Por exemplo, para exportar um filme para utilização com um iPod compatível com vídeo, seleccione "Filme para iPod". Para exportar um filme optimizado para Apple TV, seleccione "Filme para Apple TV". Também é possível criar filmes para iPhone, seleccionando uma das opções de "Filme para iPhone".
  5. Seleccione um nome de ficheiro e uma localização, e clique em Guardar.

Nota: para utilizar as mesmas definições utilizadas durante a última exportação, seleccione "Especificações mais recentes" a partir do menu instantâneo Usar.
