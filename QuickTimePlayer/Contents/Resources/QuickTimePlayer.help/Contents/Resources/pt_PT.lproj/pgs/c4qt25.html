Como criar filmes optimizados para o iPhone e a web

É possível criar facilmente versões de um filme optimizadas para visualização no iPhone e na secretária. Assim, torna-se mais simples criar filmes para distribuição online e terá a certeza de que a sua audiência obtém automaticamente a melhor experiência de visualização, de acordo com o dispositivo que utiliza.

Para exportar para iPhone e a web:

  1. Seleccione Ficheiro > Exportar para a Web.
  2. Na caixa de diálogo que aparece, especifique um nome de ficheiro e seleccione uma localização para os filmes exportados e o ficheiro de fragmento HTML.
  3. Seleccione as versões que pretende criar.
    • iPhone: Cria um filme optimizado para envio para iPhone através de uma ligação Wi-Fi.
    • iPhone (telemóvel): Cria um filme optimizado para envio para iPhone através de uma ligação de telemóvel (EDGE).
    • Secretária: Cria um filme optimizado para envio para computadores através de ligações de banda larga.
  4. Seleccione uma opção de fotograma de cartaz para o filme.

    O fotograma de cartaz determina a imagem estática que os espectadores vêem quando carregam a página da web com o filme de referência incorporado. Se pretender definir um novo fotograma de cartaz antes de exportar as versões, clique em Cancelar. Em seguida, navegue até ao fotograma que pretende utilizar e seleccione Visualização > Definir fotograma de cartaz.

  5. Clique em Exportar para criar os ficheiros necessários para as opções que seleccionou.

    O processo de exportação cria vários ficheiros, incluindo ficheiros para cada versão do filme que seleccionou, um filme de referência que irá automaticamente enviar a versão correcta para o espectador e um ficheiro HTML que contém um fragmento de código que pode utilizar para incorporar o filme numa página da web. Todos estes ficheiros têm de ser copiados para o seu servidor web, na mesma localização uns relativamente aos outros e com os mesmos nomes de ficheiros, para que o filme de referência e o fragmento HTML funcionem correctamente.
