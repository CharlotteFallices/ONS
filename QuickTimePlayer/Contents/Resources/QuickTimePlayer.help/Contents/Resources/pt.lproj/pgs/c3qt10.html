Como colar figuras e texto em um filme

Você pode colar figuras e texto em um filme do QuickTime.

Se uma figura colada for maior que as dimensões do filme, o QuickTime Player redimensiona o filme para que se ajuste às dimensões da figura. Para obter os melhores resultados, altere o tamanho das figuras (utilizando um programa gráfico) para que coincidam com as dimensões do filme, antes de inseri-las. Você pode exportar um quadro do filme para usar como guia para o posicionamento do texto (consulte Como criar uma imagem estática a partir de um filme).

Para descobrir as dimensões de um filme, escolha Janela > Mostrar Propriedades do Filme, selecione a trilha de vídeo e clique em Ajustes de Visual.

Para adicionar figuras ou texto:

  1. Copie o texto ou figura para a Área de Transferência.
  2. No QuickTime Player, mova o cursor de reprodução até o ponto em que deseja que apareça o item copiado.
  3. Escolha Editar > Colar.

Este procedimento insere a figura no filme na localização para colar, em seu próprio quadro. Para colar a figura no próprio quadro da localização para colar, escolha Editar >"Adicionar ao Filme".

Para fazer com que uma figura apareça em mais de um quadro, selecione o trecho do filme no qual deseja adicioná-la e escolha Editar > "Adicionar à Seleção e Redimensionar". É possível usar este método para adicionar o logotipo de uma empresa em todo o filme (consulte também Como alterar a transparência de uma trilha.)
