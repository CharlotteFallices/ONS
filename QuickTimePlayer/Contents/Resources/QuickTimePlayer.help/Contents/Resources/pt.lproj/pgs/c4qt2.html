Como exportar arquivos usando pré-ajustes

O QuickTime Player oferece diversos ajustes e opções para comprimir vídeo e áudio durante a exportação. Para simplificar o processo de compressão e exportação, o QuickTime Player também oferece pré-ajustes para exportação. Existem diferentes pré-ajustes para cada formato de arquivo.

Para exportar um filme usando pré-ajustes:

  1. No QuickTime Player, abra o filme que deseja exportar.
  2. Escolha Arquivo > Exportar.
  3. Escolha um formato de arquivo a partir do menu local Export.
  4. Escolha o pré-ajuste que atende melhor às suas necessidades no menu local Use.

    Por exemplo, para exportar um filme para ser usado com iPod com suporte a vídeo, escolha "Filme para iPod". Para exportar um filme otimizado para a Apple TV, escolha "Filme para Apple TV". Você também pode criar filmes para o iPhone escolhendo uma das opções "Filme para iPhone".
  5. Escolha um nome de arquivo e sua localização, e clique em Salvar.

Nota: para usar os mesmos ajustes da última exportação, escolha Ajustes Mais Recentes no menu local Usar.
