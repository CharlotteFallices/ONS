Como alterar as preferências de reprodução

Você pode alterar as preferências de reprodução escolhendo QuickTime Player > Preferências. As opções disponíveis incluem:

  • Abrir filmes em novos reprodutores:  selecione para abrir filmes em uma nova janela; desmarque para substituir o filme na janela atual pelo novo filme.
  • Reproduzir filme automaticamente ao abri-lo:  selecione para fazer com que todos os filmes sejam reproduzidos quando abertos; desmarque para fazer com que o filme inicie apenas quando você clicar no botão Reproduzir.
  • Usar ajuste de vídeo de alta qualidade quando disponível:  quando esta opção estiver selecionada, o vídeo tem mais nitidez e é reproduzido de maneira mais contínua, mas é usado mais poder de processamento.
  • Reproduzir som somente no reprodutor que estiver em primeiro plano:  selecione para reproduzir o áudio de apenas uma janela QuickTime Player (a janela ativa); desmarque para reproduzir simultaneamente o som de todos os filmes sendo reproduzidos.
  • Reproduzir som quando o aplicativo estiver em segundo plano:  selecione para reproduzir o áudio mesmo quando o QuickTime não é o aplicativo ativo; desmarque para remover o áudio quando o QuickTime não for o aplicativo ativo.
  • Mostrar equalizador:  marque ou desmarque para mostrar ou ocultar o equalizador (que mostra a presença de áudio) na janela do QuickTime Player.
  • Mostrar Guia de Conteúdo automaticamente:  selecione para ver as listas de conteúdo mais recentes quando o QuickTime Player for aberto pela primeira vez (a menos que você abra um arquivo de filme específico).
  • Colocar filmes em pausa antes de trocar de usuário:  selecione para colocar automaticamente em pausa um filme quando você usar a troca rápida de usuário para permitir que outra pessoa use seu computador.
  • Ocultar indicadores de seleção quando a seleção estiver vazia:  selecione para remover os indicadores de seleção do controle de tempo quando nenhum quadro estiver selecionado.
  • Mostrar legendas opcionais quando estiverem disponíveis:  selecione para ver informações de legenda, caso legendas tenham sido fornecidas pelos criadores do filme.
