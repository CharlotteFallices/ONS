Como alterar os níveis de volume de trilhas de áudio e outras configurações de áudio

Você pode alterar o volume e o balanço de trilhas de áudio e música. Por exemplo, se um filme tem mais de uma trilha de áudio, é possível ajustar o volume de uma trilha em relação a outra. Você também pode alterar os níveis de baixos e agudos.

Para alterar as configurações para uma trilha de áudio:

  1. No QuickTime Player, escolha Janela > Mostrar Propriedades do Filme.
  2. Na janela Propriedades, selecione a trilha de áudio e clique em Ajustes de Áudio.
  3. Arraste os controles deslizantes para definir o ajuste.
  4. Use o menu local à direita de cada canal de áudio para especificar a direção para a qual o som do canal deve ser direcionado.

    Para direcionar o som de um canal para uma caixa de auto-falante para sons baixos, escolha Tela LFE. Para especificar uma saída de áudio para o dispositivo, escolha uma das opções de Desconectado. Para silenciar um canal, escolha Não Usado.

  5. Para desativar o áudio (sem ter que apagar a trilha), selecione Mudo. Para reproduzir apenas a trilha de áudio selecionada, selecione Solo.
  6. Para salvar o balanço e os ajustes do canal com o seu filme, escolha Arquivo > Salvar ou Salvar Como.
