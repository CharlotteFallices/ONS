Como criar filmes de referência para otimizar a reprodução web

Você pode utilizar filmes de referência para fornecer o filme adequado segundo a velocidade de conexão do usuário. Por exemplo, você pode transmitir em tempo real um filme menor para usuários com um modem padrão, e uma versão maior do filme para usuários conectados por uma conexão de banda larga.

A Apple oferece uma ferramenta grátis, MakeRefMovie X, que simplifica muito o processo de criar filmes de referência. Você pode fazer o download deste software em Apple Developer Connection.
