Opções de exportação de formatos de arquivos 3G

Depois de escolher Arquivo > Exportar e clicar em Opções, os dois primeiros itens no menu local Formato do Arquivo são os formatos padrão.

  • O 3GPP funciona com redes GSM. Com esse formato, é possível usar:
    • Vídeo MPEG-4, H.263 ou H.264
    • Áudio AAC ou AMR
    • Texto temporizado 3G
  • O 3GPP2 funciona com redes CDMA 2000. Com esse formato, é possível usar:
    • Vídeo MPEG-4, H.263 ou H.264
    • Áudio AAC, AMR ou QCELP
    • Texto temporizado 3G
    • Fragmentos de filmes (permite que a reprodução de filmes mais longos comece rapidamente, já que apenas o fragmento, não o filme completo, precisa ser descarregado para o handset)

Os outros formatos são fornecidos para redes específicas. Com esses formatos, é possível restringir a distribuição para que um arquivo não possa ser compartilhado a partir do telefone do usuário (defina essa opção no painel Avançado). Esses formatos podem limitar o tamanho dos arquivos ou a velocidade de transferência de dados aceitáveis; consulte o provedor do serviço para obter mais informações.

  • 3GPP (Mobile MP4) funciona com o serviço 3G "i-motion" da NTT DoCoMo. Com este formato, você pode usar as opções 3GPP descritas acima.
  • 3GPP2 (EZmovie) funciona com o serviço de rede 3G da KDDI. Este formato oferece as mesmas opções do 3GPP2 (descrito acima), exceto pelo suporte ao áudio AMR.
  • AMC (EZmovie) funciona com o serviço de assinantes da KDDI que tenham telefones com suporte a AMC. Com esse formato, é possível usar:
    • Vídeo MPEG-4
    • Áudio QCELP
    • Formato de texto KDDI
