Como preparar um filme para transmissão em tempo real

Você pode configurar um filme para ser transmitido através da Internet. Para isso, você deve comprimir o filme para que a taxa de dados seja apropriada para a largura de banda que você pretende utilizar para a transferência.

O formato de transferência sinalizada serve para ser usado com o QuickTime Streaming Server. Ao escolher a transferência sinalizada, "trilhas de sinalização" (com as instruções necessárias para transmitir o filme em tempo real) são adicionadas ao filme. Se o filme já está no formato desejado, é possível prepará-lo para a transmissão em tempo real escolhendo Arquivo > Exportar e então "Filme para Filme Sinalizado". Caso queira mudar o formato do filme, siga os passos abaixo.

Para configurar um filme para transmissão em tempo real pela Internet:

  1. No QuickTime Player, escolha Arquivo > Exportar.
  2. Escolha "Filme para Filme do QuickTime" no menu local Exportar.
  3. Escolha uma opção de transmissão em tempo real no menu local Use.
  4. Clique em Opções e selecione a opção "Preparar para Transmissão pela Internet".
  5. Escolha Transmissão Sinalizada no menu local.

Se você deseja transmitir um filme QuickTime usando o QuickTime Streaming Server, você deve comprimir o arquivo do filme por meio de um compressor de vídeo ou de áudio especial para conteúdos Web. Todos os compressores do QuickTime são compatíveis com o QuickTime Streaming Server, mas os compressores listados aqui são otimizados para oferecer os melhores resultados na transmissão de dados pela Internet. Para obter uma lista completa dos compressores compatíveis com o QuickTime, consulte a área Produtos do site do QuickTime na web (www.apple.com/br/quicktime).

  • Compressores de vídeo: H.264, MPEG-4, Sorenson Video (qualquer versão), H.263, H.261
  • Compressores de áudio: AAC, QUALCOMM PureVoice, AMR, QDesign
