Como especificar idiomas das trilhas

Você pode criar um filme com suporte para vários idiomas, ao especificar um idioma para cada trilha. Você pode, por exemplo, criar um filme que contenha uma trilha de áudio separada para cada idioma. O QuickTime reproduz automaticamente a trilha correta, conforme o idioma padrão do computador do usuário.

Para especificar idiomas:

  1. Crie ou agrupe as trilhas alternadas.

    Grave narrações em cada um dos idiomas desejados ou crie trilhas de texto para usá-las como legendas. Salve cada uma das trilhas alternativas como um filme independente, além de colocar todas as trilhas alternativas em uma única pasta.

  2. Coloque uma cópia de seu filme original (trilha independente menos todas as trilhas que agora estão armazenadas como alternativas) nas pasta com as trilhas alternativas.
  3. Abra o filme e escolha Janela > Mostrar Propriedades do Filme.
  4. Na janela de propriedades, selecione uma trilha de som ou texto e clique em Outros Ajustes.
  5. Escolha o idioma dessa trilha no menu local Language.
  6. Escolha uma trilha a ser usada quando um idioma diferente for selecionado no menu local Alternar.
  7. Salve como filme autônomo (escolha Arquivo > Salvar Como e certifique-se de que a opção "Fazer filme autônomo" esteja selecionada).
  8. Repita estes passos para trilhas de idiomas adicionais, escolhendo, a cada vez, o idioma selecionado anteriormente no menu local Alternar.
