Como preparar um filme para início rápido

Você pode configurar um filme para que ele comece a ser reproduzido a partir de um servidor web antes de que tenha sido completamente transferido para o disco rígido do usuário. Esse tipo de filme é conhecido como filme de "Início Rápido".

Defina o ajuste Início Rápido imediatamente antes de distribuir seu filme; outras alterações podem desfazer o ajuste Início Rápido.

Se o filme estiver no formato desejado, é possível configurá-lo para o Início Rápido escolhendo Arquivo > Salvar Como e, em seguida, selecionando "Salvar Como Filme Autônomo". Se o filme não estiver no formato desejado, será necessário codificá-lo antes, seguindo os passos abaixo.

Para converter um filme e configurá-lo para o Início Rápido:

  1. No QuickTime Player, escolha Arquivo > Exportar.
  2. Escolha "Filme para Filme do QuickTime" no menu local Exportar.
  3. Clique em Opções e selecione as opções adequadas de compressão de vídeo e som adequadas para a distribuição pela web.

    Para obter mais informações, consulte Como personalizar os ajustes de exportação de vídeo, Como personalizar os ajustes de exportação de áudio.

  4. Certifique-se de que a opção "Preparar para Transmissão pela Internet" esteja selecionada e Início Rápido apareça no menu local.
