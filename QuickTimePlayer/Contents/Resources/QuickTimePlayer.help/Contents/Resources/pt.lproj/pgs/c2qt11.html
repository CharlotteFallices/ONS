Como criar uma imagem estática a partir de um filme

Com o QuickTime Player, você pode exportar um quadro único de um filme como um arquivo de imagem.

Para salvar uma imagem estática a partir de um filme:

  1. Mova o cursor de reprodução para o quadro que você deseja exportar.
  2. Escolha Arquivo > Exportar.
  3. Escolha "Filme para Imagem" no menu local Exportar.
  4. Clique em Opções e escolha um formato no menu local "Tipo de Compressão".

Para exportar cada quadro do filme como uma série de imagens, escolha "Filme para Sequência de Imagem". A exportação de cada quadro pode gerar um número muito grande de arquivos; talvez você queira exportar para uma pasta e não para a mesa.
