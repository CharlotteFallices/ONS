Como especificar estilos de fonte para trilhas de textos

Você pode alterar os atributos de fonte de uma trilha de texto contida em um filme. Para fazer isso, exporte a trilha, faça as alterações e importe a trilha novamente.

Para alterar os estilos de fonte da trilha de texto:

  1. No QuickTime Player, escolha Arquivo > Exportar.
  2. Escolha "Texto a Texto" no menu local Export.
  3. Clique em Opções e selecione "Mostrar Texto, Descritores e Hora".
  4. Use um editor de texto para alterar os valores após a etiqueta {QTtext}.
  5. Importe a trilha de texto novamente para o filme.

Você também pode inserir etiquetas ao longo de todo o texto para alterar valores enquanto o filme é reproduzido.
