Como criar filmes otimizados para o iPhone e para a web

Você pode facilmente criar versões do seu filme otimizadas para o iPhone e para a exibição no computador. Isso economiza vários passos ao criar filmes para a distribuição on-line e assegura que seu público terá automaticamente a melhor experiência de acordo com o dispositivo que estiver usando.

Para exportar para o iPhone e para a web:

  1. Escolha Arquivo > Exportar para Web.
  2. Na caixa de diálogo que aparece, especifique um nome de arquivo e selecione uma localização para os filmes exportados e para o arquivo HTML de referência.
  3. Selecione as versões que deseja criar.
    • iPhone:   cria um filme otimizado para ser visualizado no iPhone, através de uma conexão Wi-Fi.
    • iPhone (celular):   cria um filme otimizado para ser visualizado no iPhone, através de uma conexão celular (EDGE).
    • Desktop:   cria um filme otimizado para ser visualizado em computadores com conexões banda larga.
  4. Escolha um fotograma pôster para o filme.

    O fotograma pôster determina a imagem estática que será exibida ao carregar a página web com o filme de referência inserido. Caso queira definir um novo fotograma pôster antes de exportar as versões, clique em Cancelar. Navegue então até o fotograma que deseja usar e escolha Visualizar > Definir Fotograma Pôster.

  5. Clique em Exportar para criar os arquivos necessários às opções selecionadas.

    O processo de exportação cria vários arquivos, inclusive arquivos para cada versão selecionada do filme, um filme de referência que automaticamente fornecerá ao usuário a versão correta e um arquivo HTML que contém um trecho de código que pode ser usado para inserir o filme em uma página web. Todos esses arquivos devem ser copiados para o servidor web, na mesma localização relativa e com os mesmos nomes de arquivo para que o filme de referência e o código HTML funcionem corretamente.
