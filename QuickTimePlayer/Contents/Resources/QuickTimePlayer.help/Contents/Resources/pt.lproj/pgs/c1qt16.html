Ajuda QuickTime Player

Como reproduzir vários filmes de uma só vez

Você pode abrir mais de uma janela do QuickTime Player de uma só vez.

Para abrir mais de uma janela do QuickTime Player, realize uma das seguintes ações:

  • Clique duas vezes em outro arquivo de filme.
  • Escolha Visualizar > Reproduzir Todos os Filmes.

Por padrão, ao abrir mais de uma janela do QuickTime Player, o áudio é reproduzido apenas na janela ativa (que estiver à frente) do QuickTime Player. Para obter informações sobre como alterar este ajuste, consulte Como alterar as preferências de reprodução.
