QuickTime Player 輔助說明

共享影片以用於 Podcast Producer

使用 QuickTime Player 和 Mac OS X，您可以製作影片來和 Podcast Producer（Mac OS X Server v10.5 Leopard 及以上版本隨附的應用程式）搭配使用。

注意：此選項僅在您的電腦有安裝 Podcast Capture 時才能使用。

若要共享影片到 Podcast Producer：

  1. 選擇“檔案”>“共享”。
  2. 按一下 Podcast Producer。
  3. 輸入您的使用者名稱、密碼和伺服器名稱，然後按“登入”一下。
  4. 從彈出式選單中選擇 Podcast Producer 工作流程。

    請參閱 Podcast Producer Administration Guide（Podcast Producer 管理指南）或是聯絡您的網路管理者，以取得這些設定的相關資訊。
  5. 請按“共享”一下。
