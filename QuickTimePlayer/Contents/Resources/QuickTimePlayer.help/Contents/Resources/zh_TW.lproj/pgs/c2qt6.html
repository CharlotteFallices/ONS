QuickTime Player 輔助說明

錄製音訊和視訊

若有視訊攝影機和 Macintosh 電腦，您便可使用 QuickTime Player 來錄製影片。QuickTime Player 可以從大部分配備了 FireWire 的來源擷取視訊，其中包括 Apple iSight、DV 攝影機和某些網路攝影機。若有 Macintosh 或 Windows 電腦，您也可以透過內建或外接麥克風來錄製音訊。

若要使用 QuickTime Player 來錄製音訊或視訊：

  1. 將攝影機或麥克風連接到電腦（除非您使用的是內建麥克風）。
  2. 在 QuickTime Player 7 中，選擇“檔案”>“新增影片錄製”（或“檔案”>“新增音訊錄製”）。

    然後會有一個預覽視窗出現，在開始錄製前，您可以調整攝影機位置、光度和音量大小等設定。

  3. 若要開始錄製，請按視窗底部的“錄製”按鈕一下。
  4. 若要停止錄製，請再按此按鈕一下。

您錄製的影片會出現在一個 QuickTime 影片視窗裡。依照預設，此影片會儲存在桌面上，但您可以在 QuickTime Player 的“偏好設定⋯”的“錄製”面板裡選擇其他預設位置。若要查看影片的儲存位置，請按 Command 鍵並同時按 QuickTime Player 視窗上方的圖像一下。
