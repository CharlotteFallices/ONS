QuickTime Player 輔助說明

在第二個顯示器上播放影片

若電腦配備兩部顯示器，且支援雙顯示器的設定，便可以在一部顯示器上播放影片，同時在另一部顯示器上執行其他應用程式。這種設置讓您的播放進行得更加順利。

請確定沒有選擇“同步影像顯示器”。（打開“系統偏好設定”，按“顯示器”一下，並按一下“排列方式”。若看不到“排列方式”按鈕，則您的電腦並不支援此功能）。

若要在第二個顯示器上播放影片：

  1. 在 QuickTime Player 裡打開影片。
  2. 選擇“顯示方式”>“放映影片⋯”。

    兩個顯示器會並列顯示；其中一個會含有選單列。而將要用來播放影片的顯示器上會有個“Q”。若要在另外一台顯示器上播放影片，請按它一下。

  3. 從彈出式選單裡選擇一個選項。

    若要播放影片（不顯示 QuickTime Player 控制項目），請選擇“全螢幕”。

  4. 選擇“影片”或“幻燈片秀”模式。

    若您選擇幻燈片秀模式，只要按滑鼠一下，或按下向右或向左鍵便能向前或向後播放影格。

  5. 按“播放”一下。

您可以在影片播放時打開其他應用程式。按 Command + Tab 便可啟用其他的應用程式。現在您可以在影片繼續播放的同時，使用滑鼠和鍵盤來操控另一個應用程式。

若要在影片結束前停止影片，請按 Command + Tab 來啟用 QuickTime Player，然後按滑鼠一下。

所有的 QuickTime 影片播放快速鍵都能夠在全螢幕模式下使用。
