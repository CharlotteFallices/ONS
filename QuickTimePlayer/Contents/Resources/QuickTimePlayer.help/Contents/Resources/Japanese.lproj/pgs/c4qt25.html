iPhone および Web 配信に最適化されたムービーを作成する

iPhone やデスクトップ表示に最適化されたさまざまなバージョンのムービーを簡単に作成できます。これにより、オンライン配信のムービーを作成する手順がいくつか省略され、視聴者は自動的に、自分が使っている装置に最適な状態で視聴できるようになります。

iPhone および Web 配信用に書き出すには：

  1. 「ファイル」＞「Web 用に書き出す」と選択します。
  2. 表示されるダイアログで、ファイル名を指定し、書き出すムービーと HTML のスニペットファイルの場所を選択します。
  3. 作成したいバージョンを選択します。
    • 「iPhone」：Wi-Fi 接続を介した iPhone への配信に最適化されたムービーを作成します。
    • 「iPhone（セルラー）」：セルラー（EDGE）接続を介した iPhone への配信に最適化されたムービーを作成します。
    • 「デスクトップ」：ブロードバンド接続を介したコンピュータへの配信に最適化されたムービーを作成します。
  4. ムービーのポスター・フレーム・オプションを選択します。

    ポスターフレームによって、リファレンスムービーが埋め込まれた Web ページを読み込むときに視聴者に表示される静止イメージが決まります。バージョンを書き出す前に新しいポスターフレームを設定したい場合は、「キャンセル」をクリックします。次に、使用したいフレームに移動して、「表示」＞「ポスターフレームを設定」と選択します。

  5. 「書き出し」をクリックして、選択したオプションに必要なファイルを作成します。

    書き出しのプロセスによって、選択したムービーの各バージョンのファイル、視聴者に自動的に適切なバージョンを配信するリファレンスムービー、ムービーを Web ページに埋め込むために使用できるスニペットコードを含む HTML ファイルなど、いくつかのファイルが作成されます。リファレンスムービーと HTML スニペットが正しく機能するためには、これらのすべてのファイルを Web サーバ上の相対的に同じ場所に、同じファイル名でコピーする必要があります。
