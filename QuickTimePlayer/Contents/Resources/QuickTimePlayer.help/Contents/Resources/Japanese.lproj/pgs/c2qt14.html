QuickTime Player ヘルプ

ムービーを Podcast プロデューサーで共有する

「QuickTime Player」と Mac OS X を使って、Mac OS X Server v10.5 Leopard 以降に含まれているアプリケーションである「Podcast プロデューサー」で使用するムービーを作成できます。

注記：このオプションは、コンピュータに「Podcast キャプチャ」アプリケーションがインストールされている場合のみ利用できます。

ムービーを「Podcast プロデューサー」で共有するには：

  1. 「ファイル」＞「共有」と選択します。
  2. 「Podcast プロデューサー」をクリックします。
  3. ユーザ名、パスワード、およびサーバ名を入力して、「ログイン」をクリックします。
  4. ポップアップメニューから「Podcast プロデューサー」のワークフローを選びます。

    これらの設定については、「Podcast プロデューサーの管理」を参照するか、ネットワーク管理者に問い合わせてください。
  5. 「共有」をクリックします。
