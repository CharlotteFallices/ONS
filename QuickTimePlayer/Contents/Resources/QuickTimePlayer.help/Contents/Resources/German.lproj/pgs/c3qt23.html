Ändern des Filmsteuerungstyps

Sie können angeben, welche Steuerelemente für die Wiedergabe im QuickTime-Player-Fenster verfügbar sein sollen. Bei einem QTVR-Film können Sie z. B. den Film mit einer der QTVR-Filmsteuerungen sichern.

Gehen Sie wie folgt vor, um einen Filmsteuerungstyp anzugeben:

  1. Wählen Sie "Fenster" > "Filmeigenschaften einblenden".
  2. Wählen Sie "Film" und klicken Sie auf "Präsentation".
  3. Wählen Sie eine Option aus dem Einblendmenü "Art der Filmsteuerung" aus.
  4. Sichern Sie den Film.
