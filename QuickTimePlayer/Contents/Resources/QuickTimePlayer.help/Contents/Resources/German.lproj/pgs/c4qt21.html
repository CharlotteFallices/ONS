Erweiterte 3G-Optionen

Wenn Sie einen Film in das 3G-Format exportieren (durch Auswahl von "Ablage" > "Exportieren" und anschließend "Film -> 3G"), erhalten Sie Zugriff auf die folgenden Optionen. Dazu klicken Sie auf "Optionen" und anschließend auf "Erweitert".

Wenn Ihre Datei im Format "Mobile MP4" oder "EZmovie" vorliegt, können Sie die Verteilung einschränken, sodass die Datei nicht mehr weiter gesendet oder kopiert werden kann, nachdem sie auf ein Handset geladen worden ist. Sie können festlegen, wie oft die Datei nach dem Laden auf dem Handset abgespielt werden kann. Alternativ können Sie auch ein Ablaufdatum festlegen, sodass sich die Datei nach einer bestimmten Anzahl von Tagen oder ab einem bestimmten Datum nicht mehr abspielen lässt.

Die Option "Film fragmentieren" bietet die Möglichkeit, die Datei über HTTP in kleinen Einheiten zu laden, sodass die Wiedergabe schneller starten kann. Große Dateien können auf diese Weise auch auf dem Handset abgespielt werden (nur das Fragment, nicht der gesamte Film muss jeweils auf das Handset passen).
