3G-Streaming-Optionen

Wenn Sie einen Film in das 3G-Format exportieren (durch Auswahl von "Ablage" > "Exportieren" und anschließend "Film -> 3G"), erhalten Sie Zugriff auf die folgenden Optionen. Dazu klicken Sie auf "Optionen" und anschließend auf "Streaming".

Markieren Sie die Option "Streaming aktivieren", um eine Datei für das RTSP-Streaming auf den QuickTime-Player zu generieren. Mit dieser Option wird eine Steuerspur (Anweisungen, die zum Streamen einer Datei erforderlich sind) erstellt.

Die Option "Für Server optimieren" unterstützt den Server dabei, die Datei schneller zu verarbeiten, erhöht jedoch die Dateigröße.
