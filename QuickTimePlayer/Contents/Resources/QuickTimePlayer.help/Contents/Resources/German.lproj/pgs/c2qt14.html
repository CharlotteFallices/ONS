QuickTime-Player-Hilfe

Freigabe von Filmen für das Programm "Podcast-Produzent"

Mit QuickTime-Player und Mac OS X können Sie einen Film für die Verwendung mit dem Programm "Podcast-Produzent" erstellen, das mit Mac OS X Server Version 10.5 Leopard (und neuer) geliefert wird.

Hinweis: Diese Option ist nur verfügbar, wenn das Programm "Podcast-Produzent" auf Ihrem Computer installiert ist.

Gehen Sie wie folgt vor, um einen Film für das Programm "Podcast-Produzent" freizugeben:

  1. Wählen Sie "Ablage" > "Senden".
  2. Klicken Sie auf "Podcast-Produzent".
  3. Geben Sie Ihren Benutzernamen, das Kennwort und den Servernamen ein und klicken Sie dann auf "Anmelden".
  4. Wählen Sie einen Arbeitsablauf für das Programm "Podcast-Produzent" aus dem Einblendmenü aus.

    Nähere Informationen zu diesen Einstellungen finden Sie im Handbuch Podcast-Produzent Administration. Sie können sich auch an Ihren Netzwerkadministrator wenden.
  5. Klicken Sie auf "Senden".
