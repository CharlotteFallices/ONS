QuickTime-Player-Hilfe

Arbeiten mit Spuren

Ein QuickTime-Film besteht aus einer oder mehreren Spuren. Jede Spur dient zum Sichern einer bestimmten Art von Daten. QuickTime-Audio umfasst Musikspuren, die MIDI- oder andere Daten enthalten, sowie Tonspuren, die digitalisiertes Audiomaterial enthalten.

Mit QuickTime-Player, können Sie Spuren einzeln bearbeiten. Sie können Spuren von einem Film in einen anderen bewegen, aktivieren bzw. deaktivieren, entfernen oder die in einer Spur enthaltenen Informationen bearbeiten.

Weitere Informationen hierzu finden Sie unter folgenden Themen:

  • Aktivieren oder Deaktivieren von Spuren
  • Extrahieren, Hinzufügen und Bewegen von Spuren
  • Hinzufügen einer Tonspur zu einem QuickTime-Film
  • Ändern der Lautstärke einer Tonspur und anderer Audioeinstellungen
  • Hinzufügen einer Textspur in einem QuickTime-Film
  • Angeben eines Zeichensatzes für Textspuren
  • Festlegen der Anzeigedauer von Text in einer Textspur
  • Festlegen von Sprachen für einzelne Spuren
