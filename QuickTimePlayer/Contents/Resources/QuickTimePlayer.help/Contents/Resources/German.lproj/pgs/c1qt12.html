Ändern der Wiedergabeeinstellungen

Sie können die Einstellungen für die Wiedergabe ändern. Dazu wählen Sie "QuickTime-Player" > "Einstellungen". Folgende Optionen sind verfügbar:

  • Filme in einem neuen Fenster öffnen:  Wählen Sie diese Option, wenn Filme in einem neuen Player-Fenster geöffnet werden sollen. Deaktivieren Sie die Option, wenn der Film im aktuellen Player-Fenster durch den neuen Film ersetzt werden soll.
  • Filme beim Öffnen automatisch abspielen:  Wählen Sie diese Option, damit das Abspielen aller Filme sofort nach dem Öffnen gestartet wird. Deaktivieren Sie die Option, wenn der Film erst nach Klicken auf die Start/Stopp-Taste gestartet werden soll.
  • Falls verfügbar hohe Videoqualität verwenden:  Ist diese Option ausgewählt, werden Videos schärfer und homogener angezeigt, allerdings wird auch mehr Prozessorkapazität benötigt.
  • Ton nur für aktiven Player abspielen:  Wählen Sie diese Option, damit der Ton nur für ein QuickTime-Player-Fenster gespielt wird (das aktive Fenster). Deaktivieren Sie die Option, wenn der Ton aller laufenden Filme gleichzeitig gespielt werden soll.
  • Ton im Hintergrund abspielen:  Wählen Sie diese Option, damit der Ton auch dann abgespielt wird, wenn QuickTime nicht das im Vordergrund befindliche Programm ist. Deaktivieren Sie die Option, damit der Filmton abgeschaltet wird, wenn QuickTime nicht das aktive Programm ist.
  • Equalizer einblenden:  Durch Aktivieren/Deaktivieren dieser Option können Sie den Equalizer im QuickTime-Player-Fenster ein- bzw. ausblenden (dieser zeigt das Vorhandensein von Audiomaterial an).
  • HotPicks automatisch einblenden:  Wählen Sie diese Option, damit beim erstmaligen Öffnen von QuickTime-Player die neusten Inhaltsauflistungen angezeigt werden (sofern Sie keine spezifische Filmdatei öffnen).
  • Filme vor dem Benutzerwechsel anhalten:  Wählen Sie diese Option, damit ein Film automatisch angehalten wird, wenn Sie einen schnellen Benutzerwechsel durchführen, damit eine andere Person Ihren Computer verwenden kann.
  • Auswahlanzeige für leere Auswahl ausblenden: Wählen Sie diese Option, um die Auswahlanzeige in der Timeline zu entfernen, wenn keine Bilder ausgewählt sind.
  • Erweiterte Untertitel anzeigen (falls verfügbar):  Wählen Sie diese Option, damit Untertitel eingeblendet werden, sofern solche vom Autor des Films bereitgestellt wurden.
