QuickTime Player Hjälp

Dela filmer för Podcast Producer

Med QuickTime Player och Mac OS X kan du skapa en film som ska användas med Podcast Producer, ett program som ingår i Mac OS X Server 10.5 Leopard och senare.

Anmärkning: Detta alternativ är endast tillgängligt om du har Podcast Capture installerat på datorn.

Så här delar du en film till Podcast Producer:

  1. Välj Arkiv > Dela.
  2. Klicka på Podcast Producer.
  3. Ange ditt användarnamn, lösenord och servernamn och klicka sedan på ”Login”.
  4. Välj ett Podcast Producer-abetsflöde i popupmenyn.

    Se Podcast Producer Administration Guide, eller fråga nätverksadministratören, om du behöver information om dessa inställningar.
  5. Klicka på Dela.
