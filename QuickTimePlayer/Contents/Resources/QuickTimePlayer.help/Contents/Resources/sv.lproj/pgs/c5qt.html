QuickTime Player Hjälp

Automatisera QuickTime Player med AppleScript

I Mac OS X kan du automatisera funktioner i QuickTime Player med hjälp av AppleScript. Du kan till exempel öppna en film och spela upp den i ett visst antal sekunder, automatisera konverteringen av filmer från ett format till ett annat, justera uppspelningsegenskaper som starttid, volym och lager för spår eller ändra upphovsrättsinformation eller vem som publicerat filmen.

Om du vill veta vilka AppleScript-kommandon du kan använda för att automatisera QuickTime Player, så installerar du Applescript och läser QuickTime Player-ordlistan.

  1. Öppna Manusredigerare.
  2. Välj Arkiv > Öppna ordlista.
  3. Välj QuickTime Player.
