Параметры экспорта файла в формат 3G

После того, как Вы выбрали «Файл» > «Экспортировать» и нажали «Параметры», первые два элемента всплывающего меню «Формат файла» являются стандартными форматами.

  • Формат 3GPP используется для GSM-сетей. С этим форматом Вы можете использовать:
    • Видеофайлы MPEG-4, H.263 или H.264
    • Аудиофайлы AAC или AMR
    • Синхронизированный текст в формате 3G
  • Формат 3GPP2 используется для сетей CDMA 2000. С этим форматом Вы можете использовать:
    • Видеофайлы MPEG-4, H.263 или H.264
    • Аудиофайлы AAC, AMR или QCELP
    • Синхронизированный текст в формате 3G
    • Фрагменты фильмов (поддерживает более раннее воспроизведение для длинных фильмов, поскольку мобильные телефоны могут воспроизводить только фрагменты, а не целые фильмы)

Оставшиеся форматы предназначены для специальных сетей. С помощью этих форматов Вы можете ограничить распространение таким образом, что файл нельзя будет передавать с телефона получателя (эта настройка устанавливается в пункте «Дополнительно»). Эти форматы позволяют ограничить размер файл или скорость данных. Дополнительную информацию по этому вопросу можно получить у провайдера.

  • Формат 3GPP (Мобильный MP4) предназначен для сервиса NTT DoCoMo «i-motion» 3G. С помощью этого формата Вы можете использовать настройки 3GPP, описанные выше.
  • Формат 3GPP2 (EZmovie) поддерживает сетевые сервисы KDDI 3G. Данный формат предоставляет те же настройки, что и 3GPP2 (описанные выше) за исключением поддержки аудиофайлов формата AMR.
  • Формат AMC (EZmovie) используется для подписчиков KDDI с телефонами, поддерживающими AMC. С этим форматом Вы можете использовать:
    • Видеофайлы формата MPEG-4
    • Аудиофайлы формата QCELP
    • Текстовый формат KDDI
