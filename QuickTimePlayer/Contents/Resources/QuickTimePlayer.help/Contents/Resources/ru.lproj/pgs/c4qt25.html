Создание фильмов, оптимизированных для iPhone и Интернета

Можно легко создавать версии фильма, оптимизированные для просмотра на iPhone и персональных компьютерах. Это избавит Вас от необходимости выполнения нескольких шагов по созданию фильмов для распространения онлайн и автоматически обеспечит получение Вашей аудиторией наилучшего возможного качества воспроизведения видео для используемых устройств.

Чтобы экспортировать для доставки на iPhone и в Интернет:

  1. Выберите «Файл» > «Экспортировать для Интернета».
  2. В появившемся диалоге укажите имя файла и выберите папку для экспортированных фильмов и файла с фрагментом HTML.
  3. Выберите версии, которые Вы хотите создать.
    • iPhone:   Создает фильм, оптимизированный для доставки на iPhone посредством связи Wi-Fi.
    • iPhone (мобильный):   Создает фильм, оптимизированный для доставки на iPhone посредством мобильной связи (EDGE).
    • Рабочий стол:   Создает фильм, оптимизированный для доставки на компьютеры посредством широкополосной связи.
  4. Выберите один из параметров кадра-афиши для фильма.

    Кадр-афиша определяет, какое изображение будут видеть зрители при загрузке веб-страницы со встроенным ссылочным фильмом. Если Вы хотите настроить новую кадр-афишу перед экспортом версий, нажмите «Отменить». Перейдите к кадру, который Вы хотите использовать, и выберите «Вид» > «Настроить кадр-афишу».

  5. Нажмите «Экспорт», чтобы создать файлы, необходимые для выбранных Вами параметров.

    Во время экспорта создаются несколько файлов, включая файлы для каждой выбранной Вами версии фильма, ссылочный фильм, который автоматически доставляет зрителю правильную версию, а также HTML-файл, содержащий фрагмент кода, который Вы можете использовать, чтобы встроить фильм в веб-страницу. Все эти файлы должны быть копированы в Ваш веб-сервер, в одну и ту же папку и с одинаковыми именами файлов. Это обеспечит правильную работу ссылочного фильма и фрагмента HTML.
