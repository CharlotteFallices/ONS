Справка QuickTime Player

Обмен фильмами по электронной почте или через Интернет

С помощью QuickTime Player в Mac OS X можно легко создавать фильмы и обмениваться ими с другими пользователями по электронной почте или в MobileMe. Выберите способ обмена фильмом, и QuickTime Player автоматически создаст файл для этого способа доставки. (При использовании функции «Общий доступ» для размещения фильмов в Сети требуется учетная запись MobileMe.)

Для обмена фильмами:

  1. Выберите «Файл» > «Общий доступ».
  2. Нажмите e-mail или HomePage.
  3. Введите имя и выберите размер фильма.
  4. Если Вы собираетесь разместить фильм в Интернете, можете впечатать название фильма (оно появится на веб-странице).
  5. Нажмите «Общий доступ».

Во время экспорта фильма можно нажать на значок «x» для отмены экспорта.

Если Вы нажали e-mail, откроется окно нового сообщения в Mail с вложенным фильмом. При нажатии HomePage браузер откроет веб-страницу с фильмом. Отправьте URL друзьям, чтобы они могли посмотреть фильм.
