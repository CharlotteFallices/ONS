Справка QuickTime Player

Общий доступ к фильмам для Podcast Producer

С помощью QuickTime Player и Mac OS X можно создать фильм для использования с Podcast Producer, программой, являющейся частью Mac OS X Server 10.5 Leopard и более поздних версий.

Примечание. Данный параметр доступен только в случае, если на Вашем компьютере установлена программа «Podcast Capture».

Для предоставления общего доступа к фильму в программе «Podcast Producer»:

  1. Выберите «Файл» > «Общий доступ».
  2. Нажмите «Podcast Producer».
  3. Введите Ваше имя пользователя, пароль и имя сервера, а затем нажмите «Вход».
  4. Выберите процесс Podcast Producer во всплывающем меню.

    См. Podcast Producer Administration Guide или обратитесь к Вашему сетевому администратору для получения информации об этих настройках.
  5. Нажмите «Общий доступ».
