Справка QuickTime Player

Подготовка фильма к потоковой передаче в режиме реального времени

Программа позволяет настроить потоковую передачу фильма по сети Интернет. Для этого Вы должны выполнить сжатие фильма, чтобы скорость передачи данных соответствовала пропускной способности подключения других пользователей.

«Хинтованный поток» передачи используется для серверов QuickTime Streaming или Darwin Streaming. При выборе «Хинтованный поток» к фильму добавляются «дорожки подсказки» (информация, необходимая для потоковой передачи фильма). Если фильм уже находится в нужном формате, Вы можете подготовить его к потоковой передаче, выбрав «Файл» > «Экспортировать», а затем указав «Видеофильм – Hinted Movie». Если Вы хотите изменить формат фильма, выполните следующие действия.

Чтобы настроить фильм для потоковой передачи через Интернет:

  1. В меню QuickTime Player выберите «Файл» > «Экспортировать».
  2. Во всплывающем меню «Экспортировать» выберите «Видеофильм – Фильм QuickTime».
  3. Выберите параметры потока во всплывающем меню «Использовать».
  4. Нажмите «Параметры» и установите флажок перед «Подготовить для интернет-трафика».
  5. Выберите во всплывающем меню «Хинтованный поток»

Если Вы хотите передать фильм QuickTime через сервер QuickTime Streaming, Вы должны использовать оптимизированную для сети систему сжатия аудио и видеофайлов. Все системы сжатия QuickTime совместимы с сервером QuickTime Streaming, однако системы, перечисленные далее специально оптимизированы для получения оптимального результата при передаче файлов через Интернет. Полный список алгоритмов сжатия, которые поддерживает QuickTime, см. на странице «Продукты» веб-сайта QuickTime (www.apple.com/ru/quicktime).

  • Видеокомпрессоры: H.264, MPEG-4, Sorenson Video (любая версия), H.263, H.261
  • Аудиокомпрессоры: AAC, QUALCOMM PureVoice, AMR, QDesign
