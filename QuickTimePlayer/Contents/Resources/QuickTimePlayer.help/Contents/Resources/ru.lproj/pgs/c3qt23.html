Изменение типа панели управления

Программа позволяет выбирать тип элементов управления воспроизведением в окне QuickTime Player. Например, в фильме QTVR можно сохранить фильм с помощью одной из панелей управления.

Чтобы выбрать тип панели управления:

  1. Выберите «Окно» > «Показать свойства фильма».
  2. Выберите строку с фильмом и нажмите «Презентация».
  3. Выберите параметр во всплывающем меню «Тип панели управления».
  4. Сохраните фильм.
