Справка QuickTime Player

Экспорт файлов с использованием предварительных настроек

QuickTime Player предлагает множество различных функций и настроек для сжатия аудио- и видеофайлов во время экспорта. Для упрощения процесса сжатия и экспорта QuickTime Player предоставляет набор предварительных настроек. Для каждого формата файлов предусмотрен свой набор предварительных настроек.

Чтобы экспортировать фильм, используя предварительные настройки:

  1. В QuickTime Player откройте фильм, который Вы хотите экспортировать.
  2. Выберите «Файл» > «Экспортировать».
  3. Во всплывающем меню «Экспортировать» выберите формат файла.
  4. Выберите настройку, наиболее отвечающую Вашим требованиям, во вплывающем меню «Установки».

    Например, чтобы экспортировать фильм для использования на iPod, поддерживающем видео, выберите «Фильм в iPod». Чтобы экспортировать фильм, оптимизированный для Apple TV, выберите «Фильм на Apple TV». Вы также можете создать фильмы для iPhone, выбрав один из параметров «Фильм на iPhone».
  5. Укажите название файла и его расположение и нажмите «Сохранить».

Примечание. Чтобы использовать те же настройки, что и при последней операции экспорта, выберите пункт «Последние настройки» во всплывающем меню «Использовать».
