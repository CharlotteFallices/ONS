Справка QuickTime Player

Изменение уровня громкости и других настроек звуковых дорожек

Программа позволяет изменять громкость и баланс аудио и музыкального сопровождения. Например, если в фильме содержится более одной аудиодорожки, можно настроить громкость дорожек по отношению одна к другой. Вы также можете изменить уровень высоких и низких частот.

Чтобы изменить настройки аудиодорожек:

  1. В меню QuickTime Player выберите «Окно» > «Показать свойства фильма».
  2. В окне «Свойства» выберите аудиодорожку и нажмите «Аудио».
  3. С помощью бегунков установите необходимые настройки.
  4. Для указания направления звука канала воспользуйтесь всплывающим меню, расположенным справа от каждого аудиоканала.

    Для направления звука в сабвуфер выберите выберите «LFE Screen». Чтобы указать определенный выход для аудиоустройства, выберите один из «Отдельный». Для отключения канала выберите пункт «Свободный».

  5. Для отключения аудиодорожки (без удаления этой дорожки) укажите «Выключить звук». Для воспроизведения только указанной аудиодорожки укажите «Соло».
  6. Для сохранения настроек баланса и каналов Вашего фильма выберите «Файл» > «Сохранить» или «Сохранить как».
