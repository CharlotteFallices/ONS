Справка QuickTime Player

Настройка параметров аудиофайлов

Чтобы задать настройки экспорта аудиофайлов:

  1. Выберите «Файл» > «Экспортировать».
  2. Во всплывающем меню «Экспортировать» выберите «Видеофильм – Фильм QuickTime».
  3. Нажмите «Параметры».
  4. В части «Звук» в диалоговом окне «Параметры фильма» нажмите «Параметры» и задайте необходимые параметры.

    «Формат»: выберите компрессор (кодек), который Вы хотите использовать для компрессии звука. Для стандартных типов использования и скачивания звука через Интернет рекомендуется использовать MPEG-4 Audio (AAC).

    «Каналы»: выберите между моно (1 канал) или стерео (2 канала).

    «Интенсивность»: цифровой звук состоит из оцифрованных звуковых образцов. Чем выше частота образцов в секунду, тем лучше качество звука. Для обеспечения высокого качества при воспроизведении музыки требуется более высокая частота модуляции, чем для разговорной речи, поскольку в музыке используется больший диапазон частот. В аудио-CD используется частота модуляции 44,1 кГц.

    Настройки визуализации:  выберите качество обработки сигналов, которую будете использовать. Настройки для более высокого качества требуют больше времени на их обработку.

    Другие параметры:  для настройки дополнительных специальных параметров для выбранного кодека, или чего-либо другого, нажмите «Показать дополнительные настройки», если данная кнопка возможна для выбора.
