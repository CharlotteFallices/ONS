Справка QuickTime Player

Подготовка фильмов к воспроизведению через Интернет

Программа позволяет создавать фильмы, которые можно отправлять по сети Интернет. Два способа передачи фильмов через Интернет:

  • При загрузке HTTP фильм загружается на жесткий диск пользователя. «Быстрый старт»» – это функция QuickTime, которая позволяет пользователю смотреть или слушать файлы до окончания загрузки (задолго до окончания загрузки всего фильма) фильма со стандартного веб-сервера на жесткий диск. «Быстрый старт» хорошо подходит для короткометражных фильмов с ограниченным размером файла. Эта функция обеспечивает высокое качество воспроизведения вне зависимости от скорости подключения к Интернету, хотя при использовании медленных соединений пользователю придется дольше ждать момента запуска файла.
  • Потоковая передача в режиме реального времени через сервер QuickTime Streaming Server (QTSS) обеспечивает передачу фильмов в режиме реального времени как при модемном, так и при широкополосном доступе в Интернет. Файлы не загружаются на жесткий диск пользователя. Воспроизведение файла (но не сохранение) производится программным обеспечением пользователя в процессе загрузки. При просмотре прямых трансляций через Интернет, загрузке длинных видеофильмов, прослушивании радиопередач через Интернет и телевизионных каналов и в других ситуациях, когда сохранение файлов на жесткий диск не требуется, рекомендуется использовать потоковую передачу в режиме реального времени, а не функцию «Быстрый старт». В QTSS используется протокол RTSP.
