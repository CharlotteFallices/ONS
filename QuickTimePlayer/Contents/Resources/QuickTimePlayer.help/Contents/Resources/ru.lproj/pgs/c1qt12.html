Изменение настроек режима воспроизведения

Вы можете изменить настройки режима воспроизведения, выбрав QuickTime Player > «Настройки». В данном меню представлены следующие параметры:

  • Показывать каждый фильм в новом плеере:  установите флажок, чтобы открывать каждый фильм в новом окне, или снимите флажок – новый файл откроется в текущем окне.
  • Автоматически начинать показ фильма при открытии:  воспроизведение фильмов начинается автоматически сразу же после открытия файла; снимите флажок с настройки – воспроизведение начнется только после нажатия на кнопку «Воспр.».
  • По возможности используйте настройки высококачественного видео:  при выборе этой настройки повышается качество видео при воспроизведении, однако, при этом затрачивается большее количество ресурсов процессора.
  • Включать звук только в плеере на переднем плане:  установленный флажок включает воспроизведение аудиофайла только в одном окне QuickTime-плеера (активное окно), при снятом флажке – звук воспроизводится во всех воспроизводимых фильмах.
  • Включать звук, если программа в фоновом режиме:  установите флажок для воспроизведение звука при работе QuickTime в фоновом режиме или снимите флажок, чтобы аудио не воспроизводилось, если QuickTime не является активной программой.
  • Показывать эквалайзер:  поставьте или снимите флажок, чтобы показать или скрыть эквалайзер (указывающий на наличие аудио) в окне QuickTime-плеера.
  • Автоматически показывать Content Guide:  выберите эту настройку для показа списка контента при первом запуске программы «QuickTime-плеер» (если только Вы не открываете специальный файл с фильмом).
  • Останавливать показ перед переключением между пользователями:  автоматически приостанавливать показ фильма при быстром переключении пользователей для предоставления другому пользователю доступа к компьютеру.
  • Скрыть индикаторы выбора для пустого выбранного фрагмента:  выберите для удаления индикаторов выделения из временной шкалы, если ни один кадр не выбран.
  • По возможности показывать скрытые субтитры:  выберите для просмотра информации в скрытых субтитрах, если они были предусмотрены производителем фильма.
