Справка QuickTime Player

Создание фильмов со ссылками (reference movies) для оптимизации воспроизведения через Интернет

Вы можете использовать фильмы со ссылками для оптимизации воспроизведения фильмов с учетом различной скорости соединения компьютеров. Например, можно передавать фильм меньшего объема для пользователей со стандартным соединением через модем или более объемную версию для пользователей с широкополосным соединением.

Apple предоставляет бесплатный инструмент MakeRefMovie X, который позволяет значительно упростить процесс создания фильмов со ссылками (reference movies). Это программное обеспечение можно загрузить через Apple Developer Connection.
