Справка QuickTime Player

Автоматизация задач QuickTime Player с помощью программы AppleScript

Для автоматизации функций QuickTime Player в Mac OS X Вы можете использовать программу AppleScript. Например, Вы можете открыть фильм и задать воспроизведение в течение определенного отрезка времени; автоматически выполнить преобразование фильмов их одного формата в другой; настроить такие параметры воспроизведения дорожек, как время запуска, громкость и слой, либо установить свойства фильма, как авторское право и создатель.

Чтобы узнать обо всех командах AppleScript, которые Вы можете использовать для управления QuickTime Player, установите AppleScript и воспользуйтесь словарем AppleScript.

  1. Откройте программу «Редактор скриптов».
  2. Выберите «Файл» > «Открыть список функций».
  3. Из списка выберите QuickTime Player.
