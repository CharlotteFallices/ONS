Справка QuickTime Player

Настройки экспорта видеофайлов формата 3G

  • Формат видео: Если в исходном фильме только одна видеодорожка и она уже сжата, выберите «Пропустить», чтобы избежать повторного сжатия.
  • Скорость: Чем выше количество передаваемых килобит информации в секунду (Кбит/с), тем выше качество фильма. Однако для получения оптимального качества воспроизведения не указывайте скорость данных выше доступной пропускной способности.
  • Оптимизация: Выберите «H.264» во всплывающем меню «Формат видео», затем укажите предпочтительный метод отправки во всплывающем меню «Оптимизация». Этот параметр указывает кодеку, насколько может отличаться скорость передачи данных от указанного Вами значения.
  • Размер изображения: Если Вы отсылаете файл на мобильный телефон, выберите один из стандартов. Стандарт «Текущий» поддерживает размер исходного файла, но получившийся файл не может воспроизводиться с мобильных телефонов. Чтобы выбрать размер, не указанный во всплывающем меню, выберите параметр «Произвольный»».
  • Сохранять формат изображения с помощью: Если Вы изменяете размер изображения, используйте этот параметр для указания точного значения параметра, в случае необходимости изменения масштаба фильма. Letterbox изменит масштаб источника пропорционально в размер очищенной диафрагмы, добавляя при необходимости черные полосы вверху и внизу или по бокам изображения. Функция обрезания центрует, изменяет масштаб и обрезает в соответствии с очищенной диафрагмой. Подогнать под размеры регулирует исходный размер в соответствии с размером назначения, подгоняя по размеру к более длинной стороне и при необходимости масштабируя.
  • Частота кадров: В большинстве случаев качество видеофайлов будет более высоким, если выбрать число, делящееся без остатка на значение частоты кадров исходного файла. Например, если исходный файл передавался со частотой 30 кадров в секунду, выберите частоту 10 или 15. Не указывайте частоту кадров, превышающую частоту исходного файла.
  • Ключевой кадр: Чаще всего Вы указываете ключевой кадр (с наименьшим номером), однако, чем лучше качество, тем больше размер файла.

Чтобы указать дополнительные параметры нажмите «Настройки видео»:

  • Маркеры повторной синхронизации: Добавьте маркеры повторной синхронизации внутри видеокадров, чтобы помочь при восстановлении утерянных пакетов при пакетной передаче данных.
  • Режим кодирования: При выборе видеофайла формата H.264 Вы можете ускорить процесс сжатия (к примеру, с целью просмотра), выбрав пункт «Быстр.кодир.(Однопрох.)». При выборе параметра по умолчанию «Лучш.качество (Многопрох.)» кодек определяет, сколько проходов необходимо для сжатия фильма при сохранении оптимального качества.
