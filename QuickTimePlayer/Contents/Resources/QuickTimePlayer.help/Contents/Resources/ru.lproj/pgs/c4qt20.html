Справка QuickTime Player

Настройки потока 3G

При экспорте фильмов в формат 3G («Файл» > «Экспортировать», а затем выбрав «Видеофильм – 3G») нажмите «Параметры», а затем «Поток», чтобы перейти к следующим параметрам.

Чтобы создать файл для потоковой передачи через протокол RTSP в QuickTime Player, установите флажок перед «Разрешить поток». Этот параметр позволяет создать дорожку подсказки (инструкции для потоковой передачи файла).

Настройка «Оптимизировать для сервера» помогает серверу быстрее обрабатывать файлы, однако это увеличивает размер файлов.
