QuickTime Player 帮助

通过电子邮件或 Web 共享影片

使用 QuickTime Player 和 Mac OS X，您可以轻松创建适合通过电子邮件或 MobileMe HomePage 与他人分享的影片。只要选择您想要共享影片的方式，QuickTime Player 会自动创建适用于该共享方式的文件。（要使用“共享”功能将影片放到 Web 上，您需要有 MobileMe 帐户。）

要共享影片：

  1. 选取“文件”>“共享”。
  2. 点按“电子邮件”或 HomePage。
  3. 选取影片的名称和大小。
  4. 如果您要在 Web 上发布影片，则可以键入影片的名称（您键入的名称将显示在网页上）。
  5. 点按“共享”。

影片正在导出时，如有必要，您可以点按“x”以取消导出。

如果点按“电子邮件”，Mail 将打开一个新的信息窗口，其中附加有您的影片。如果点按 HomePage，浏览器将打开嵌入了该影片的特定网页。将该 URL 发送给朋友们，他们就可以观看该影片。
