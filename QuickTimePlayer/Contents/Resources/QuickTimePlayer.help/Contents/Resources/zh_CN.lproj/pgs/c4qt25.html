创建优化用于 iPhone 和 Web 传输的影片

您可以轻松创建用于在 iPhone 和台式机上观看的各种优化版本的影片。这能够为您省去创建影片用于在线分发的若干步骤，并且确保了您的观众将根据他们所用的设备而自动获得最好的观看体验。

要导出以用于 iPhone 和 Web 传输：

  1. 选取“文件”>“导出给 Web”。
  2. 在出现的对话框中，指定文件名称，并为导出的影片和 HTML 片段文件选择位置。
  3. 选择您想要创建的版本。
    • iPhone：    创建为通过 Wi-Fi 连接传输到 iPhone 而优化的影片。
    • iPhone（蜂窝电话）：    创建为通过蜂窝电话 (EDGE) 连接传输到 iPhone 而优化的影片。
    • 桌上型电脑：    创建为通过宽带连接传输到电脑而优化的影片。
  4. 为影片选取标志帧选项。

    标志帧决定了观众在载入包含嵌入式参考影片的网页时将看到的静止图像。如果您想要先设定新的标志帧再导出影片版本，请点按“取消”。然后，导航到您想要使用的帧，选取“显示”>“设定标志帧”。

  5. 点按“导出”以便为您选定的选项创建所需的文件。

    导出过程会创建几个文件，包括您选择的每个影片版本的文件、会自动将正确版本传输给观众的参考影片，以及包含用于将影片嵌入到网页中的代码段的 HTML 文件。必须将所有这些文件拷贝到您的 Web 服务器，放置在相对彼此都相同的位置，并使用相同的文件名称，这样参考影片和 HTML 片段才能正常工作。
