QuickTime Player 帮助

与 Podcast Producer 共享影片

使用 QuickTime Player 和 Mac OS X，您可以创建影片与 Podcast Producer 配合使用，Podcast Producer 是 Mac OS X Server v10.5 Leopard 或更高版本系统附带的一款应用程序。

注：只有当您在电脑上安装了 Podcast Capture 应用程序时，此选项才可用。

要将影片共享到 Podcast Producer：

  1. 选取“文件”>“共享”。
  2. 点按“Podcast Producer”。
  3. 输入您的用户名称、密码和服务器名称，然后点按“登录”。
  4. 从弹出式菜单中选取一个 Podcast Producer 工作流程。

    请参阅《Podcast Producer Administration Guide》（Podcast Producer 管理指南），或者咨询您的网络管理员，了解有关这些设置的信息。
  5. 点按“共享”。
