Cómo crear películas optimizadas para usarse con el iPhone y para su distribución web

Puede crear fácilmente versiones de su película optimizadas para verlas en un iPhone y en el ordenador. De este modo, se ahorrará varios pasos del proceso de creación de películas para su distribución on-line y se asegurará de que los destinatarios de la película reciban automáticamente la versión más adecuada para el dispositivo que estén usando.

Para exportar una película para usarse con el iPhone y para su distribución web:

  1. Seleccione Archivo > Exportar para web.
  2. En el cuadro de diálogo que aparecerá, especifique un nombre de archivo y seleccione una ubicación para las películas exportadas y el archivo de fragmento de código HTML.
  3. Seleccione las versiones que desee crear.
    • iPhone: crea una película optimizada para la distribución a un iPhone a través de una conexión Wi-Fi.
    • iPhone (móvil): crea una película optimizada para la distribución a un iPhone a través de una conexión de telefonía móvil (EDGE).
    • Escritorio: crea una película optimizada para la distribución a ordenadores a través de conexiones de banda ancha.
  4. Seleccione una opción de fotograma póster para la película.

    El fotograma póster determina qué imagen verán los espectadores cuando carguen la página web en la que se encuentra la película de referencia. Si desea seleccionar otro fotograma póster antes de exportar las distintas versiones, haga clic en Cancelar. A continuación, vaya hasta el fotograma que desee usar y seleccione Visualización > Fijar fotograma póster.

  5. Haga clic en Exportar para crear los archivos necesarios para las opciones que ha seleccionado.

    Durante el proceso de exportación se crearán varios archivos: los archivos de cada una de las versiones de la película que haya seleccionado, una película de referencia que permite al espectador recibir automáticamente la versión correcta, y un archivo HTML que contiene un fragmento de código que puede usar para incluir la película en una página web. Debe copiar todos estos archivos en su servidor web, con la misma estructura y con los mismos nombres de archivo, para que tanto la película de referencia como el código HTML funcionen correctamente.
