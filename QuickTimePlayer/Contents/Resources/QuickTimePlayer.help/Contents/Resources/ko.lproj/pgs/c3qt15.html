QuickTime Player 도움말

개별 트랙에 대한 언어 지정하기

개별 트랙의 언어를 지정하여 다중 언어를 지원하는 동영상을 생성할 수 있습니다. 예를 들어, 각 언어에 대한 개별 오디오 트랙을 포함하는 동영상을 생성할 수 있습니다. QuickTime은 사용자 컴퓨터의 언어 설정을 기반하여 자동으로 올바른 트랙을 재생합니다.

언어를 지정하려면,

  1. 대체 트랙을 생성하거나 모으십시오.

    원하는 언어로 음성을 씌우거나 부제목으로 사용할 텍스트 트랙을 생성하십시오. 각 대체 트랙을 독립된 동영상으로 저장하고 하나의 폴더에 함께 저장하십시오.

  2. 독립된 원본 동영상과 대체물로 현재 저장되어 있는 모든 이하 트랙의 복사본을 대체 폴더에 넣으십시오.
  3. 동영상을 열고 윈도우 > 동영상 속성 보기를 선택하십시오.
  4. 속성 윈도우에서, 사운드 또는 텍스트 트랙을 선택하고 기타 설정을 클릭하십시오.
  5. 언어 팝업 메뉴에서 이 트랙의 언어를 선택하십시오.
  6. 대체 팝업 메뉴에서 다른 언어 선택 시 사용할 트랙을 선택하십시오.
  7. 독립된 동영상으로 저장하십시오.(파일 > 별도 저장을 선택하고 "독립된 동영상 만들기"가 선택되어 있는지 확인하십시오.)
  8. 추가의 언어 트랙에 대해 이 단계를 반복하십시오. 매번 대체 팝업 메뉴에서 이전에 선택된 언어를 선택하십시오.
