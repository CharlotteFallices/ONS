QuickTime Player 도움말

재생 환경설정 변경하기

QuickTime Player > 환경설정을 선택하여 재생 환경설정을 변경할 수 있습니다. 사용할 수 있는 옵션이 다음을 포함합니다.

  • 새로운 플레이어로 동영상 열기:  선택하면 동영상이 새로운 플레이어 윈도우로 열리며, 선택 해제하면 현재 플레이어 윈도우의 동영상이 새로운 동영상으로 대치됩니다.
  • 열릴 때 자동으로 동영상 재생:  선택하면 모든 동영상이 열리자 마자 재생되며, 선택 해제하면 재생 단추를 클릭하면 동영상이 시작됩니다.
  • 사용 가능할 때 고화질 비디오 설정 사용:  이 옵션이 선택되어 있으면, 비디오가 더 선명하고 부드럽게 보이지만 더 많은 프로세서 파워가 소모됩니다.
  • 전면에 있는 플레이어에서만 사운드 재생:  선택하면 하나의 QuickTime Player 윈도우(활성 윈도우)에 대해서만 오디오가 재생되며, 선택 해제하면 재생 중인 모든 동영상의 사운드가 동시에 재생됩니다.
  • 응용 프로그램이 후면에 있을 때 사운드 재생:  선택하면 QuickTime이 맨 앞에 있는 응용 프로그램이 아닐 때도 오디오가 재생되며, 선택 해제하면 QuickTime이 활성 중인 응용 프로그램이 아닐 때에는 오디오의 소리가 꺼집니다.
  • 이퀄라이저 보기:  선택하거나 선택 해제하면 QuickTime Player 윈도우에 이퀄라이저(오디오의 존재를 나타냄)가 나타나거나 가려집니다.
  • 자동으로 콘텐츠 가이드 보기:  선택하면 QuickTime Player가 처음으로 열릴 때 최신 콘텐츠 목록이 나타납니다(특정 동영상 파일을 열지 않는 이상).
  • 사용자를 전환하기 전에 동영상 일시 정지:  선택하면 신속한 사용자 전환을 사용하여 다른 사람이 자신의 컴퓨터를 사용할 수 있도록 할 때 동영상을 자동으로 일시 정지시킵니다.
  • 빈 선택 항목에 대한 선택 표시 가리기: 아무 프레임도 선택되어 있지 않을 때 타임라인에서 선택 표시를 제거하려면 선택하십시오.
  • 사용 가능할 때 청각 장애인용 자막 보기:  동영상 제작자가 자막을 제공하는 경우 청각 장애인용 자막 정보를 보려면 선택하십시오.
