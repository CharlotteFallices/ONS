프레임 번호 및 시간 코드 사용하기

QuickTime Player에서 동영상을 볼 때 현재 동영상 시간, 프레임 번호 또는 동영상에 시간 코드 트랙이 있는 경우에는 SMPTE 시간 코드를 표시할 수 있습니다. (Final Cut Pro와 같은 일부 응용 프로그램은 캡처 중에 시간 코드를 자동으로 추가합니다.)

프레임 번호, 동영상 시간 또는 시간 코드 표시하기

QuickTime Player에서 타임라인 옆의 영역에 현재 동영상에 대한 시간 정보가 표시됩니다.

  1. 시간 표시 영역을 클릭하십시오.
  2. 나타나는 팝업 메뉴에서 옵션을 선택하십시오.

    표준: HH:MM:SS 포맷으로 동영상 시간을 표시합니다.

    시간 코드: HH:MM:SS:프레임 포맷으로 동영상의 시간 코드를 표시합니다. 동영상의 초당 프레임 수에 따라 시간 코드나 드롭 프레임 시간 코드를 입력하십시오. 시간 표시 영역에서 프레임 부분에 있는 콜론은 비 드롭 프레임이 표시 중임을 나타냅니다. 세미콜론은 드롭 프레임 시간 코드임을 나타냅니다. 동영상에 시간 코드 트랙이 있다면 이것이 사용됩니다. 그렇지 않은 경우에는 시간 코드가 동영상의 시작에서 00:00:00:00부터 계산됩니다.

    프레임 번호: 동영상의 시작에서 0부터 증가되는 프레임 번호가 표시됩니다. 시간 표시 영역에 있는 "f"는 프레임 수가 표시됨을 나타냅니다.
