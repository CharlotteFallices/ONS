Aide QuickTime Player

Changement du type de barre de commandes associé à une séquence

Vous pouvez choisir le type de commandes de lecture qui seront disponibles dans la fenêtre QuickTime Player. Dans le cas d’une séquence QTVR, par exemple, vous pouvez enregistrer la séquence avec l’un des contrôleurs de séquences adaptés à QTVR.

Pour définir un type de contrôleur de séquences :

  1. Choisissez Fenêtre > Propriétés de la séquence.
  2. Sélectionnez Séquence puis cliquez sur Présentation.
  3. Choisissez une option dans le menu local Type de contrôleur de séquences.
  4. Enregistrez la séquence.
