Présentation d’une séquence sous forme de diaporama

Dans un diaporama, le spectateur doit cliquer pour passer à l’image suivante. Vous pouvez lire une séquence sous forme de diaporama.

Pour lire une séquence sous forme de diaporama :

  1. Choisissez Présentation > Présenter la séquence.
  2. Cliquez sur Diaporama, puis cliquez sur Lecture.
  3. Pour avancer dans le diaporama, cliquez sur le bouton de la souris ou bien appuyez sur les flèches gauche ou droite.
