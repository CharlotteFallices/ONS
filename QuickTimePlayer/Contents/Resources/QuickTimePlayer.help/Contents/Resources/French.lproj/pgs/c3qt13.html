Spécification des styles de police pour les pistes de texte

Vous pouvez modifier les polices de caractères d’une piste de texte déjà incluse dans une séquence. Pour cela, vous devez exporter la piste, effectuer vos modifications, puis réimporter la piste.

Pour changer la police de la piste de texte :

  1. Dans QuickTime Player, choisissez Fichier > Exporter.
  2. Dans le menu local Exporter, choisissez « Texte vers texte ».
  3. Cliquez sur Options et sélectionnez « Afficher le texte, les descriptions et l’heure ».
  4. Utilisez un éditeur de texte pour modifier les valeurs après la balise {QTtext}.
  5. Importez de nouveau la piste texte dans votre séquence.

Vous pouvez également insérer des balises dans le texte pour modifier les valeurs pendant la lecture de la séquence.
