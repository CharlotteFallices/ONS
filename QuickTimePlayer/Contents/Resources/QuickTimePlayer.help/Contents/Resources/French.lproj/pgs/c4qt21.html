Options 3G avancées

Lorsque vous exportez une séquence au format 3G (en choisissant Fichier > Exporter puis en sélectionnant « Séquence vers 3G »), vous avez accès aux options suivantes en cliquant sur Options puis en cliquant sur Avancé.

Si votre fichier est au format Mobile MP4 ou EZmovie, vous pouvez en restreindre la distribution. Ainsi, une fois le fichier envoyé sur un téléphone portable, il est alors impossible de le transférer ou de le copier. Vous pouvez définir le nombre de fois que le fichier peut être lu sur le téléphone portable une fois téléchargé. Il est également possible de rendre le fichier inutilisable au bout d’un certain nombre de jours ou après une certaine date.

L’option Fragmenter la séquence permet de fragmenter le fichier avant de le télécharger via HTTP afin d’accélérer la lecture sur le téléphone portable (à chaque fois, le fragment, et non la séquence entière doit être mémorisé dans le téléphone).
