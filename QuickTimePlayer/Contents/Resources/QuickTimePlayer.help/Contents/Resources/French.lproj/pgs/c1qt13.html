Aide QuickTime Player

Visionnage de séquences en mode plein écran

Vous pouvez configurer l’écran de telle sorte que seule la séquence soit visible (et non la fenêtre de QuickTime Player, le bureau ou d’autres fenêtres). Ce mode de présentation s’appelle « mode plein écran » (Vous pouvez aussi configurer la séquence pour la présenter à la moitié de sa taille d’origine, au double de sa taille, etc.).

Pour lire une séquence en mode plein écran, procédez de l’une des deux manières suivantes :

  • Choisissez Présentation > Plein écran.
  • Choisissez Présentation > Présenter la séquence, sélectionnez Plein écran puis cliquez sur Lecture.

Pour quitter le mode plein écran, appuyez sur Échap. Pour régler la manière dont l’image est affichée pour qu’elle corresponde à la taille de l’écran, en l’étirant ou en la mettant au format letterbox par exemple, déplacez le pointeur vers le haut de l’écran pour afficher les options du menu de QuickTime Player, puis choisissez un réglage dans le menu Présentation. Les options supplémentaires suivantes sont disponibles en mode plein écran :

  • Adapter à l’écran : le contenu est mis à l’échelle pour remplir l’écran du mieux possible sans pour autant rogner la séquence ni en changer les proportions.
  • Réduire/agrandir : le contenu est mis à l’échelle et rogné pour remplir complètement l’écran au moins d’un côté. Ceci élimine éventuellement les zones noires en haut (letterbox) ou sur le côté (pillarbox) mais peut masquer ou déformer le contenu.
  • Panoramique : le contenu est mis à l’échelle pour remplir l’écran et ses bords horizontaux extérieurs sont compressés afin d’éviter de rogner l’image.

Vous pouvez également parcourir toutes les options disponibles en plein écran en cliquant sur le bouton dans les commandes de la séquence qui apparaissent lorsque vous déplacez le pointeur.

Vous pouvez également définir des options vous permettant d’afficher toutes les séquences en mode plein écran. Vous pouvez masquer les commandes de lecture ou modifier le délai pendant lequel elles sont visibles avant d’être masquées. Pour définir ces options, consultez les préférences QuickTime Player.

Remarque : si l’option « Afficher les contrôles en mode plein écran » est sélectionnée dans les Préférences de QuickTime Player, vous pouvez faire apparaître les commandes masquées en bougeant le pointeur de la souris.
