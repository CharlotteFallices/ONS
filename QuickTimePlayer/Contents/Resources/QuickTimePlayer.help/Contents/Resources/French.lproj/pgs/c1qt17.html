Aide QuickTime Player

Utilisation du numéro d’image et de codes temporels

Lorsque vous visualisez une séquence dans QuickTime Player, vous pouvez afficher le code temporel, les numéros d’images ou encore le code temps SMPTE de la séquence en cours si elle possède une piste de code temporel. (Certaines applications comme Final Cut Pro ajoutent automatiquement un code temporel lors de la capture.)

Affichage des numéros d’image, du temps de la séquence ou du code temporel:

La zone située à côté de la chronologie dans QuickTime Player affiche les informations relatives au temps de la séquence ouverte.

  1. Cliquez sur la zone d’affichage du temps.
  2. Choisissez une option dans le menu local qui apparaît alors.

    Standard : affiche le temps de la séquence sous la forme HH:MM:SS.

    Code temporel : affiche le code temporel de la séquence sous la forme HH:MM:SS:Image. Selon le nombre d’images par seconde de la séquence, le code temporel ou le code temporel avec perte d’images est alors affiché. Le symbole deux-points présent dans la portion du code temporel réservée aux images indique que le code temporel affiché ne tient pas compte des pertes d’images. Un point-virgule dénote un code temporel avec perte d’images. Si la séquence possède une piste de code temporel, cette dernière est utilisée. Dans le cas contraire, le code temporel est calculé à partir de 00:00:00:00 au début de la séquence.

    Numéro d’image : affiche le numéro des images, qui augmente d’une unité à chaque image en partant de 0 au début de la séquence. La lettre « f », dans la zone d’affichage du temps, indique que le décompte des images est affiché.
