Aide QuickTime Player

Partage de séquences par courrier électronique ou par le web

Avec QuickTime Player et Mac OS X, vous n’aurez aucune difficulté à créer des séquences parfaites pour être partagées avec d’autres via le courrier électronique ou votre page d’accueil MobileMe. Il vous suffit d’indiquer la façon dont vous voulez partager votre séquence et QuickTime Player crée automatiquement un fichier adapté à la méthode de transmission choisie. (Pour utiliser la fonction de partage permettant de publier votre film sur Internet, il vous faut un compte MobileMe.)

Pour partager une séquence :

  1. Sélectionnez Fichier > Partager.
  2. Cliquez sur Courrier électronique ou sur Page Web.
  3. Indiquez le nom et la taille de la séquence.
  4. Si vous comptez publier votre séquence sur le web, vous pouvez donner à la séquence un nom qui s’affichera sur la page web.
  5. Cliquez sur Partager.

Pendant l’exportation de la séquence, vous pouvez cliquez sur le « x » pour annuler l’opération si nécessaire.

Si vous avez sélectionné Courrier électronique, une fenêtre de nouveau message s’ouvre dans Mail, avec votre séquence en pièce jointe. Si vous cliquez sur Page Web, votre navigateur web s’ouvre et affiche la page web spécifiée avec la séquence incorporée. Il vous suffit ensuite d’envoyer l’URL de la page à vos amis pour qu’ils puissent visionner la séquence.
