Préparation d’une séquence pour la diffusion continue en temps réel

Vous pouvez configurer une séquence en vue d’une diffusion en continu sur Internet. Pour ce faire, vous devez compresser la séquence en adaptant le débit à la bande passante utilisée pour se connecter.

Le format d’enchaînement à indications est destiné au Serveur Enchaînement QuickTime ou au Serveur Enchaînement Darwin. Ce format permet l’ajout de « pistes indicatives » (informations nécessaires pour diffuser la séquence en continu) à la séquence. Si la séquence est déjà au format désiré, vous pouvez la préparer pour la diffuser en continu en choisissant Fichier > Exporter, puis sélectionnez « Séquence vers séquence à indications ». Si vous voulez modifier le format de la séquence, procédez comme suit.

Pour configurer une séquence pour diffusion sur Internet

  1. Dans QuickTime Player, choisissez Fichier > Exporter.
  2. Sélectionnez l’option « Séquence vers Séquence QuickTime » dans le menu local Exporter.
  3. Choisissez une option de diffusion dans le menu local Mode.
  4. Cliquez sur Options, puis cochez la case Préparer pour la diffusion en continu sur Internet.
  5. Choisissez Enchaînement à indications dans le menu local.

Si vous voulez diffuser une séquence QuickTime à l’aide du Serveur Enchaînement QuickTime, vous devez utiliser un compresseur vidéo ou audio optimisé pour compresser le fichier. Tous les compresseurs QuickTime sont compatibles avec le Serveur Enchaînement QuickTime ; ceux indiqués ci-dessus sont en outre optimisés pour donner des résultats optimaux dans le cas d’une diffusion sur Internet. Pour obtenir la liste complète des compresseurs reconnus par QuickTime, consultez la section Produits du site web de QuickTime (www.apple.com/fr/quicktime).

  • Compresseurs vidéo : H.264, MPEG-4, Sorenson Video (n’importe quelle version), H.263, H.261
  • Compresseurs audio : AAC, QUALCOMM PureVoice, AMR, QDesign
