Aide QuickTime Player

Partage de séquences pour Podcast Producer

Avec QuickTime Player et Mac OS X, vous pouvez créer une séquence à utiliser avec Podcast Producer, une application incluse dans Mac OS X Server 10.5 Leopard et ultérieur.

Remarque : cette option est disponible uniquement si vous avez installé l’application Capture de podcast sur votre ordinateur.

Pour partager une séquence vers Podcast Producer :

  1. Sélectionnez Fichier > Partager.
  2. Cliquez sur Podcast Producer.
  3. Saisissez votre nom d’utilisateur, votre mot de passe et le nom du serveur, puis cliquez sur Ouvrir une session.
  4. Choisissez un flux de production de Capture de podcast dans le menu local.

    Consultez le Guide d’administration de Podcast Producer ou votre administrateur réseau pour obtenir des informations sur ces réglages.
  5. Cliquez sur Partager.
