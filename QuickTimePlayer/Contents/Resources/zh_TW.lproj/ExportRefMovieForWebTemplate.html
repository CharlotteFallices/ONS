在網站上張貼 $(TITLE) 的指示說明

您已使用“QuickTime 專業版”來輸出 $(TITLE)。下列資訊將會協助您將影片張貼到網站。

在與此教學 HTML 檔案相同的檔案夾中，“QuickTime 專業版”已放置了您影片的多種版本，適合在網頁上傳送。該檔案夾中也會有一個參考影片，其會針對您使用者的各種設備和連線速度，選擇最佳的影片版本。您可以將參考影片包含在 HTML 頁面中，藉此確保使用者能在網站上觀賞到品質較佳的影片。

請使用下列的 HTML 語法來將您的影片嵌入至網頁。

重要告示：

  • 若要讓您的媒體能夠在 Internet 上檢視，媒體內容（影片和展示影像）必須放置於可公開取用的伺服器上。
  • 如果您的媒體內容是來自您網頁以外的另一個位置，則 URL 參數必須修改。
  • 影片 URL 參數必須指向參考影片（$(REF_MOVIE_HTML_ESCAPED)）而不是來源影片。
  • 參考影片必須要與來源影片保持在相同的檔案夾中。
  • 來源影片的名稱無法更改，但您可以更改參考影片的名稱（也請記得在 HTML 語法中更改此資訊）。
  • 請注意：若要正確地將網頁驗證為 XHTML，則 <style>標籤與第一個 <script> 標籤必須放置於網頁的 <head> 標籤中。
